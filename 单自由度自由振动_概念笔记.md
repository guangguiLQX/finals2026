# 单自由度系统自由振动 · 概念笔记

> 覆盖 PPT 第03讲（无阻尼基础）、第04讲（能量法）、第05讲（阻尼自由振动）  
> 考试占比大，是后续所有章节的基石

---

## 一、这一章在讲什么？

给你一个弹簧-质量（-阻尼）系统，不给外力，只给一个**初始扰动**（拉开松手/推一下），问：

1. 它以什么**频率**振动？（→ ωₙ）
2. 它振动的**方程**是什么？（→ x(t) 表达式）
3. 如果有阻尼，振动**衰减多快**？（→ ζ, δ）

## 二、无阻尼自由振动（第03讲）

### 2.1 物理模型

```
     ┌───┐
     │ m │──→ x（坐标原点取静平衡位置）
     ├───┤
     │ k │
     └─┴─┘
   固定端
```

### 2.2 从牛顿第二定律到振动方程

**第一步**：画受力。质量块受弹簧力 `F = -kx`（负号表示力总是指向平衡位置——恢复力）。

**第二步**：牛顿第二定律 `F = mẍ`：

```
mẍ = -kx
mẍ + kx = 0        ← 无阻尼自由振动方程
```

**第三步**：标准化（两边除以 m）：

```
ẍ + ωₙ²x = 0       其中 ωₙ² = k/m
```

### 2.3 固有频率 ωₙ

```
ωₙ = √(k/m)   [rad/s]    圆频率
fₙ = ωₙ/(2π)  [Hz]       自然频率
T = 2π/ωₙ     [s]        周期
```

**这是整个课程最重要的一个公式。** ωₙ 只由系统本身的 m 和 k 决定：
- 想提高固有频率 → 加硬弹簧 或 减质量
- 想降低固有频率 → 换软弹簧 或 加质量

**重力的影响**：把坐标原点设在**静平衡位置**，重力项从方程中自动消失。所以：
> 重力改变平衡位置，不改变固有频率。

顺便记一个实用公式：`ωₙ = √(g/δ_st)`，δ_st 是静变形（挂上重物后弹簧伸长量）。

### 2.4 方程的解

`ẍ + ωₙ²x = 0` 是二阶常系数线性齐次微分方程。它的通解（就是 x(t) 的表达式）有几种等价写法：

**形式一（余弦+正弦）——代入初始条件最方便**：

```
x(t) = C₁cos(ωₙt) + C₂sin(ωₙt)
```

由初始条件 x(0)=x₀, ẋ(0)=v₀ 定常数：
```
C₁ = x₀
C₂ = v₀/ωₙ

所以：
x(t) = x₀cos(ωₙt) + (v₀/ωₙ)sin(ωₙt)
```

**形式二（单一正弦）——看振幅和相位最直观**：

```
x(t) = A·sin(ωₙt + φ)

A = √(x₀² + (v₀/ωₙ)²)      ← 振幅
φ = arctan(ωₙx₀/v₀)        ← 初相位
```

### 2.5 物理图像

```
x(t)
 ↑
A ┤     ······          ← 正弦曲线，振幅 A 永远不变
  │   ··      ··            （因为无阻尼，不消耗能量）
 0 ┤··----------··----→ t
  │  ··          ··
-A ┤    ··········
  │
  └── T = 2π/ωₙ ──┘
```

### 2.6 扭转振动（类比）

把 x 换成 θ，m 换成 I（转动惯量），k 换成 k_t（扭转刚度）：

```
Iθ̈ + k_t·θ = 0
ωₙ = √(k_t/I)
```

### 2.7 复摆（PPT例题类型）

```
I₀θ̈ + mg·a·sinθ = 0
小角度：sinθ ≈ θ
ωₙ = √(mga/I₀)
```

平行轴定理：`I₀ = I_c + ma²`（I_c 是绕质心的转动惯量，a 是质心到转轴的距离）

## 三、能量法（第04讲）——复杂系统的利器

### 3.1 为什么需要能量法？

牛顿法需要分析每个力的方向、约束关系，对复杂系统（滑轮、杠杆、多弹簧）很烦。

能量法只需要写两个标量：**动能 T** 和 **势能 V**，不用管力的方向。

### 3.2 能量守恒

无阻尼自由振动中，机械能守恒：

```
T + V = 常数
d(T+V)/dt = 0           ← 求导得到运动方程（等价于牛顿法）

T_max = V_max           ← 直接从最大值关系求 ωₙ（更简单！）
```

### 3.3 等效质量 Me 和等效刚度 Ke ——核心思想

设一个广义坐标（比如某个质量的位移 x），把系统中**所有部件的动能和势能都折算到这个坐标上**：

```
T = ½·M_eq·ẋ²        所有动能归结为一个等效质量 M_eq
V = ½·K_eq·x²        所有势能归结为一个等效刚度 K_eq
```

然后直接：

```
ωₙ = √(K_eq / M_eq)
```

### 3.4 弹簧串并联的等效刚度

```
串联（弹簧首尾相接）：
┌──[k1]──[k2]──┐     两个弹簧受力相同 → 总变形 = 分别变形之和
    1/K_eq = 1/k₁ + 1/k₂         ← 倒数相加，越串越软
    K_eq = k₁k₂/(k₁+k₂)

并联（弹簧并排连接）：
┌──[k1]──┐
├──[k2]──┤               两个弹簧变形相同 → 总力 = 分别力之和
    K_eq = k₁ + k₂              ← 直接相加，越并越硬
```

> 记法：串联像电阻并联（倒数加），并联像电阻串联（直接加）

### 3.5 常见折算套路

**情形1：通过杠杆连接**
```
    质量在杠杆一端，弹簧在另一端
    杠杆比 = a/b
    
    M_eq = m·(a/b)²           ← 杠杆比要平方！
    K_eq = k·(a/b)²
```

**情形2：滑轮+绳索**
```
    滑轮有转动惯量 I_p，半径 R
    质量 m 的线位移 = x，滑轮角位移 = x/R
    
    动能：T = ½mẋ² + ½I_p(ẋ/R)² = ½(m + I_p/R²)ẋ²
    M_eq = m + I_p/R²
    
    对于匀质圆盘滑轮：I_p = ½M_p·R²
    M_eq = m + ½M_p
```

**情形3：纯滚动圆盘/圆柱**
```
    质量 m，半径 R，做纯滚动（不打滑）
    平动速度 = v，角速度 = v/R
    
    平动动能：½mv²
    转动动能：½I_c·(v/R)²
    
    对于匀质圆柱体：I_c = ½mR²
    转动动能 = ½·½mR²·(v/R)² = ¼mv²
    
    总动能 T = ½mv² + ¼mv² = ½·(3m/2)·v²
    M_eq = 3m/2
```

**情形4：弹簧本身有质量**
```
    弹簧质量 m_t 不可忽略，一端固定一端运动
    弹簧上各点速度不同（从0线性增大到ẋ）
    积分得弹簧动能 = ½·(m_t/3)·ẋ²
    
    M_eq = m + m_t/3     ← 注意是 m_t/3，不是 m_t/2！
```

### 3.6 能量法解题标准步骤

> 这是考试大题的高频题型，步骤固定，练熟就能拿分。

1. **选广义坐标**（标在图上，比如 "以 x 为质量 m 的位移"）
2. **写动能 T**：每个有质量的部件，用 ẋ 表示其速度，写出 ½mv² 或 ½Iω²
3. **写势能 V**：每个弹簧的变形量用 x 表示，写出 ½k(变形)²；如果有重力，写 mgh
4. **提取 M_eq 和 K_eq**：`T = ½M_eqẋ²`，`V = ½K_eqx² + 常数`
5. **套公式**：`ωₙ = √(K_eq/M_eq)`

## 四、阻尼自由振动（第05讲）

### 4.1 加了阻尼器之后

```
     ┌───┐
     │ m │──→ x
     ├─┬─┤
     │k│c│      ← 阻尼器和弹簧并联（最常见配置）
     └─┴─┘
```

运动方程：
```
mẍ + cẋ + kx = 0
```

标准化（两边除以 m）：
```
ẍ + 2ζωₙẋ + ωₙ²x = 0
```

### 4.2 核心概念：阻尼比 ζ

```
ζ = c / c_c = ────── = ──────
              2√(mk)    2mωₙ
```

- **临界阻尼系数** `c_c = 2√(mk) = 2mωₙ`
- ζ 是个**无量纲数**，描述阻尼"够不够强"
- ζ < 1 → 欠阻尼（大多数工程情况）
- ζ = 1 → 临界阻尼（刚好不振荡）
- ζ > 1 → 过阻尼（太黏了，回位更慢）

### 4.3 特征方程与特征根

设解为 `x = e^(st)`，代入方程：

```
s² + 2ζωₙs + ωₙ² = 0

s₁,₂ = -ζωₙ ± ωₙ√(ζ²-1)
```

| ζ 范围 | 根的情况 | 物理行为 |
|--------|---------|---------|
| ζ < 1 | 共轭复根 | 振荡衰减 |
| ζ = 1 | 重实根 | 临界衰减 |
| ζ > 1 | 两个不等实根 | 过阻尼缓慢衰减 |

### 4.4 欠阻尼（ζ < 1）——考试只考这个

**阻尼固有频率**：
```
ω_d = ωₙ√(1-ζ²)
```

> ω_d < ωₙ（阻尼让频率略微降低），但当 ζ 较小时差距不大。  
> 例如 ζ=0.2，ω_d = 0.98ωₙ，仅降2%。

**通解**：
```
x(t) = e^(-ζωₙt) · [C₁cos(ω_d t) + C₂sin(ω_d t)]
```

代入初始条件 x(0)=x₀, ẋ(0)=v₀：

```
x(t) = e^(-ζωₙt) · [x₀cos(ω_d t) + ────────·sin(ω_d t)]
                                       ω_d
```

**物理图像**：
```
x(t)
 ↑
 │  ··                            ← 包络线 ±Ae^(-ζωₙt)
 │  ·  ··    ··    ··             ← 包络线内是 cos/sin 振荡
 │··    ····  ····  ··→ t
 │  ··                       
 │    ··  ··  ··  ··
 │
```

三个特征：
- **振荡频率** ω_d ≈ ωₙ（阻尼小）
- **振幅指数衰减**，衰减速度由 ζωₙ 决定
- **阻尼周期** `T_d = 2π/ω_d > T`（比无阻尼时稍微长一点）

### 4.5 临界阻尼（ζ = 1）

```
x(t) = e^(-ωₙt)·(C₁ + C₂t)
      = e^(-ωₙt)·[x₀ + (v₀ + ωₙx₀)t]
```

不振荡，以**最快速度**回到平衡位置。阻尼器设计目标通常是这个状态。

### 4.6 过阻尼（ζ > 1）

```
x(t) = e^(-ζωₙt)·[C₁cosh(pt) + C₂sinh(pt)]
     其中 p = ωₙ√(ζ²-1)
```

也不振荡，但回位速度**比临界阻尼慢**。阻尼越大反而越慢——这和直觉相反！

### 4.7 对数衰减率 δ——实验测阻尼比

这是连接**理论（ζ）**和**实验（测振幅）**的桥梁。

**定义**：相邻两个正峰值的比，取自然对数。

```
      ┌  xᵢ  ┐
δ = ln│──────│ = ζωₙT_d
      └ xᵢ₊₁ ┘
```

代入 T_d = 2π/(ωₙ√(1-ζ²))：

```
         2πζ
δ = ──────────
      √(1-ζ²)
```

**ζ 较小时的近似**（ζ < 0.2）：
```
δ ≈ 2πζ
ζ ≈ δ/(2π)
```

**实际测量中跨 j 个周期**（降低测量误差）：
```
      1      ┌  xᵢ  ┐
δ = ─── · ln│──────│
      j      └ xᵢ₊ⱼ ┘
```

### 4.8 对数衰减率的物理直觉

| ζ | δ = 2πζ/√(1-ζ²) | 每周期振幅衰减到 |
|---|------------------|-----------------|
| 0.01 | 0.063 | 93.9% |
| 0.05 | 0.314 | 73.0% |
| 0.10 | 0.632 | 53.2% |
| 0.20 | 1.28 | 27.7% |

> ζ=0.05 时，每周期振幅只降到约 73%，衰减很慢。  
> ζ=0.20 时，每周期只保留约 28%，衰减很快。

### 4.9 品质因数 Q 与阻尼比的关系

```
Q = 1/(2ζ)
```

在受迫振动（第06讲）里会大量用到。Q 就是共振时放大因子 H(s=1) 的值。

## 七、最容易出错的细节

1. **阻尼比 ζ 的分母有2**：`ζ = c/(2√(mk))` ← 最容易写漏
2. **ω_d = ωₙ√(1-ζ²)**，根号里是 `1-ζ²` 不是 `ζ²-1`
3. **弹簧自身质量修正 m_t/3**，不是 m_t/2
4. **杠杆比的平方**：用能量法往广义坐标折算，比例系数自动平方
5. **坐标系取静平衡位置**：此时重力项自动消失，不需要刻意处理
6. **ζ 极小时 ω_d ≈ ωₙ**，但响应表达式中 `e^(-ζωₙt)` 的衰减仍然存在
7. **串联弹簧是倒数相加** ← 和电阻并联一个规则
8. **纯滚动圆盘**的转动惯量 `I_c = ½mR²`，动能 = 平动 + 转动 = ½mv² + ½·½mR²·(v/R)² = ¾mv²
