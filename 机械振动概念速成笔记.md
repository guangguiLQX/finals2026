# 机械振动 · 概念速成笔记

> 同济车辆工程 · 2026年期末  
> 从零开始，只看考试要用的

---

## 一、什么是机械振动？

**定义**：物体在**平衡位置附近**做**往复运动**。

三个关键词：
- **平衡位置**——物体不动时待的地方（弹簧自然长度处、静平衡位置）
- **往复**——来回折腾
- **系统**——由质量（惯性元件）+ 弹簧（弹性元件）+ 阻尼器（耗能元件）组成

**为什么重要？**
- 共振会让桥塌（Tacoma大桥1940年）
- 汽车悬架要隔振（不然屁股遭罪）
- 发动机转子不平衡会抖

---

## 二、振动的分类

### 按自由度分
| 类型 | 含义 | 例子 |
|------|------|------|
| 单自由度 | 一个坐标描述运动 | 弹簧挂一个质量块 |
| 多自由度 | n个独立坐标 | 多个弹簧-质量串联 |
| 连续系统 | 无限自由度 | 梁、弦（本课不考） |

### 按激励分
| 类型 | 含义 |
|------|------|
| **自由振动** | 给一个初始扰动（拉一下放手），之后没有外力 |
| **受迫振动** | 有持续外力在推（如电机旋转产生的周期性力） |

### 按阻尼分
| 类型 | 含义 |
|------|------|
| 无阻尼 | 理想情况，振幅永不衰减 |
| 有阻尼 | 实际情况，振幅越来越小直到停下 |

### 按方程分
| 类型 | 含义 |
|------|------|
| 线性振动 | 微分方程是线性的（叠加原理成立） |
| 非线性振动 | 方程含非线性项（本课不考） |

## 三、单自由度系统的三大元件

### 3.1 质量（惯性元件）m

```
F = m · a = m · ẍ
```

- 单位：kg
- 作用：储存**动能** `T = ½mẋ²`
- 关键性质：质量越大，越难加速，固有频率越低

### 3.2 弹簧（弹性元件）k

```
F = k · x   （胡克定律：力与位移成正比）
```

- 单位：N/m（平动弹簧），N·m/rad（扭转弹簧）
- 作用：储存**势能** `V = ½kx²`
- 关键性质：刚度越大，固有频率越高

**弹簧串并联**：
```
串联（两端连在一起）：1/k_eq = 1/k₁ + 1/k₂    ← 越串越软
并联（并排连在一起）：k_eq = k₁ + k₂           ← 越并越硬
```

### 3.3 阻尼器（耗能元件）c

```
F = c · ẋ   （粘性阻尼：力与速度成正比）
```

- 单位：N·s/m
- 作用：消耗能量（把机械能变热能）
- 关键性质：阻尼越大，振动衰减越快
- **临界阻尼系数**：`c_c = 2√(mk) = 2mωₙ`

## 四、核心概念：固有频率 ωₙ

### 4.1 是什么？

**系统"天生"的振动频率**，只由 m 和 k 决定，与初始条件和外力无关。

```
ωₙ = √(k/m)   [rad/s]  圆频率（角频率）

fₙ = ωₙ/(2π)  [Hz]     自然频率（每秒振动几次）

T = 2π/ωₙ     [s]      周期（振动一次要多久）
```

### 4.2 直觉理解

- **硬弹簧（k大）+ 轻质量（m小）→ ωₙ 大 → 振动快**
- **软弹簧（k小）+ 重质量（m大）→ ωₙ 小 → 振动慢**
- 就像吉他弦：拧紧（增k）音高，换粗弦（增m）音低

### 4.3 重力的影响

重力会让弹簧有一个**静变形** `δ_st = mg/k`，但：

> ⚠️ **重力只改变平衡位置，不改变固有频率！**
> 
> 把坐标原点设在静平衡位置，方程里重力项就消掉了。

顺便：`ωₙ = √(g/δ_st)` —— 测出静变形就能得到固有频率。

## 五、自由振动

### 5.1 无阻尼自由振动

**方程**：`mẍ + kx = 0` → `ẍ + ωₙ²x = 0`

**解**：
```
x(t) = A·sin(ωₙt + φ)

A     ——振幅（振动最大偏离量）
φ     ——初相位（决定 t=0 时物体在哪）
```

由初始条件确定 A 和 φ：
```
x(0) = x₀,  ẋ(0) = v₀

A = √(x₀² + (v₀/ωₙ)²)
φ = arctan(ωₙx₀/v₀)
```

**特征**：振幅不变，永不停止（理想化模型）

### 5.2 有阻尼自由振动

**方程**：`mẍ + cẋ + kx = 0` → `ẍ + 2ζωₙẋ + ωₙ²x = 0`

**核心参数——阻尼比 ζ**：
```
ζ = c / c_c = c / (2√(mk)) = c / (2mωₙ)
```
这个**无量纲数**决定了一切行为。

### 5.3 三种阻尼情况

| 条件 | 名称 | 行为 |
|------|------|------|
| **ζ < 1** | **欠阻尼** | 振荡衰减，工程最常见 |
| ζ = 1 | 临界阻尼 | 最快回到平衡位置，不振荡 |
| ζ > 1 | 过阻尼 | 缓慢爬回平衡位置，更慢 |

**欠阻尼的解**（考试只考这个）：
```
ω_d = ωₙ√(1-ζ²)          ← 阻尼固有频率（略低于ωₙ）

x(t) = e^(-ζωₙt) · [x₀cos(ω_d t) + (v₀+ζωₙx₀)/ω_d · sin(ω_d t)]
      \_________/   \_________________________________________/
       包络线衰减              振荡部分
```

**关键图像**：振幅被 `e^(-ζωₙt)` 包住，越来越小。

### 5.4 对数衰减率 δ（实验测阻尼的方法）

```
δ = ln(x₁/x₂) = 2πζ/√(1-ζ²)

当 ζ < 0.2 时：δ ≈ 2πζ

跨 j 个周期：δ = (1/j)·ln(xᵢ/xᵢ₊ⱼ)
```

**物理意义**：相邻两次振幅比值的自然对数，表征衰减快慢。

## 六、受迫振动

### 6.1 是什么？

系统在**持续外部激励**下的振动。激励通常是周期性的力 `F(t) = F₀sin(ωt)`。

### 6.2 两个阶段

```
全响应 = 瞬态响应 + 稳态响应

瞬态（通解）：初始条件引起的自由振动，有阻尼会衰减掉
稳态（特解）：以激励频率 ω 持续振动，不衰减
```

> 考试主要考**稳态响应**。瞬态部分通常在阻尼作用下很快消失。

### 6.3 核心无量纲参数

**频率比**：
```
s = ω / ωₙ
```
- s << 1：激励比固有频率慢得多
- s = 1：**共振**
- s >> 1：激励比固有频率快得多

### 6.4 幅频特性——放大因子 H(s)

```
H(s) = ──────────────────
        √((1-s²)² + (2ζs)²)
```

**三个区的物理意义**：

| 区间 | H ≈ | 控制因素 | 物理 |
|------|-----|---------|------|
| s << 1 | 1 | 刚度 k | 响应≈静位移 F₀/k |
| s = 1 | 1/(2ζ) | 阻尼 c | 共振！阻尼越小峰值越大 |
| s >> 1 | 1/s² → 0 | 质量 m | 惯性跟不上了 |

**共振时**（s=1）：相位差 φ = 90°，位移落后力四分之一周期。

**幅值最大的频率**（精确值）：
```
s_peak = √(1-2ζ²)    ← 略小于1
H_max  = 1/(2ζ√(1-ζ²)) ≈ 1/(2ζ)   （ζ很小时）
```

### 6.5 相频特性

```
φ = arctan(──────)
            1 - s²
```

- s << 1：φ ≈ 0°（位移与力同相）
- s = 1：φ = 90°
- s >> 1：φ ≈ 180°（位移与力反相）

### 6.6 品质因数 Q

```
Q = 1/(2ζ)  = H(s=1)
```

- Q 越大 → 共振峰越尖 → 阻尼越小
- **半功率带宽法**测量：`Q = ωₙ/(ω₂-ω₁)`，其中 ω₁, ω₂ 是 H = Q/√2 时的两个频率

## 七、基础激励与隔振

### 7.1 基础激励

不是力作用在质量上，而是**基础/支撑在振动**（如地震、路面不平）。

基础运动：`x_f(t) = Dsin(ωt)`（基础位移幅值为D）

### 7.2 绝对传递率 η₂

```
           ┌                  ┐½
           │    1 + (2ζs)²    │
η₂(s) =   │ ───────────────  │
           │ (1-s²)² + (2ζs)² │
           └                  ┘
```

物理意义：**传到质量上的振动 / 基础的振动**

### 7.3 隔振的黄金法则

```
           s < √2         s = √2        s > √2
           ───────       ───────       ───────
           η₂ > 1        η₂ = 1        η₂ < 1
           放大区        临界点        隔振区 ✓
```

> ⚠️ **s = √2 是所有阻尼曲线交汇点**——不管阻尼多大，传递率在此处都是1。这是隔振设计的核心判据。

**隔振设计要点**：
- 让 `ωₙ << ω/√2`（选软弹簧降固有频率）
- 阻尼在共振区有益，在隔振区有害
- 一般选 s > 3 以获得较好隔振效果

---

## 八、不平衡质量激励

旋转机械（电机、发动机）的偏心质量产生惯性力。

等效激励力：`F₀ = meω²`（m=偏心质量，e=偏心距，ω=转速）

> ⚠️ **关键区别**：普通力激励 F₀ 是常数；不平衡激励 F₀ 随 ω² 增大。s→0 时响应趋近于 0（不是静位移！）

## 九、任意激励下的响应（时域）

### 9.1 单位脉冲响应 h(t)

在 t=0 给系统一个瞬时冲击（Dirac δ函数），系统的响应叫 h(t)：

```
h(t) = ──── · e^(-ζωₙt) · sin(ω_d t)    （欠阻尼）
       mω_d
```

h(t) 是系统在时域的"指纹"——只由 m, k, c 决定，与激励无关。

### 9.2 Duhamel 积分

任意力 F(t) 的响应 = 力和 h(t) 的卷积：

```
x(t) = ∫₀ᵗ F(τ) · h(t-τ) dτ
```

物理理解：把任意力切成无数小脉冲，每个产生一个小响应，全部加起来。

## 十、多自由度系统

### 10.1 核心思想

单自由度不够用了（比如想描述汽车前后轮两处振动），需要多个坐标。

### 10.2 运动方程的矩阵形式

```
M ẍ + K x = 0
```

- **M**（质量矩阵）—— n×n，正定，通常是对角矩阵
- **K**（刚度矩阵）—— n×n，对称，可能半正定
- **x**（位移向量）—— n×1

**刚度矩阵元素 kᵢⱼ**：第 j 个自由度发生单位位移时，第 i 个自由度需要的力。

### 10.3 特征值问题（求固有频率和振型）

```
(K - ω²M) φ = 0

非零解条件：det(K - ω²M) = 0   ← 特征方程
```

解出 n 个固有频率：`ω₁ < ω₂ < ... < ωₙ`

每个 ωᵢ 代回去得到对应**振型（模态）φ⁽ⁱ⁾**——系统以该频率振动时，各自由度振幅的比例关系。

> 模态只说**形状**（例如"两端同向"还是"两端反向"），不说具体幅值。

### 10.4 模态正交性——多自由度的"魔法"

不同阶模态之间满足：
```
(φ⁽ⁱ⁾)ᵀ M φ⁽ʲ⁾ = 0   (i≠j)
(φ⁽ⁱ⁾)ᵀ K φ⁽ʲ⁾ = 0   (i≠j)
```

定义模态质量和模态刚度：
```
mₚᵢ = (φ⁽ⁱ⁾)ᵀ M φ⁽ⁱ⁾     kₚᵢ = (φ⁽ⁱ⁾)ᵀ K φ⁽ⁱ⁾

且 ωᵢ² = kₚᵢ / mₚᵢ
```

### 10.5 模态叠加法（解耦）

令 `x = Φ p`（物理坐标 = 模态矩阵 × 模态坐标），代入 Mẍ + Kx = 0：

```
n个耦合的方程  →  n个独立的单自由度方程
mₚᵢ p̈ᵢ + kₚᵢ pᵢ = 0   (i = 1~n)
```

每个 pᵢ 都像单自由度那样解！最后 `x(t) = Σ φ⁽ⁱ⁾ pᵢ(t)` 叠加回物理响应。

**这就是整个多自由度理论的最终目的**——把复杂系统拆成一堆简单系统分别求解。

### 10.6 零根（刚体模态）

如果 `det(K) = 0`，刚度矩阵不满秩，则存在 ω = 0 的特征值。

物理意义：系统可以不做弹性变形地运动（整体平动或转动）。

对应的模态坐标解：`p(t) = at + b`（线性漂移，不是振动！）

## 十一、核心公式速查卡

### 单自由度
| 公式 | 含义 |
|------|------|
| `ωₙ = √(k/m)` | 无阻尼固有频率 |
| `ζ = c/(2√(mk))` | 阻尼比 |
| `ω_d = ωₙ√(1-ζ²)` | 阻尼固有频率 |
| `δ = 2πζ/√(1-ζ²)` | 对数衰减率 |
| `s = ω/ωₙ` | 频率比 |
| `H = 1/√((1-s²)²+(2ζs)²)` | 放大因子 |
| `φ = arctan(2ζs/(1-s²))` | 相位差 |
| `Q = 1/(2ζ)` | 品质因数 |
| `η₂ = √((1+(2ζs)²)/((1-s²)²+(2ζs)²))` | 绝对传递率 |
| `h(t) = e^(-ζωₙt)·sin(ω_d t)/(mω_d)` | 单位脉冲响应 |

### 多自由度
| 公式 | 含义 |
|------|------|
| `Mẍ + Kx = 0` | 运动方程 |
| `det(K - ω²M) = 0` | 特征方程求频率 |
| `(K - ωᵢ²M)φ⁽ⁱ⁾ = 0` | 求模态振型 |
| `(φ⁽ⁱ⁾)ᵀMφ⁽ʲ⁾ = 0` | M-正交性 |
| `(φ⁽ⁱ⁾)ᵀKφ⁽ʲ⁾ = 0` | K-正交性 |
| `x = Φp` | 模态坐标变换 |

### 能量法
| 公式 | 含义 |
|------|------|
| `T = ½M_eq·ẋ²` | 等效质量表示动能 |
| `V = ½K_eq·x²` | 等效刚度表示势能 |
| `Tₘₐₓ = Vₘₐₓ` | 能量守恒 → 求ωₙ |
| `ωₙ = √(K_eq/M_eq)` | 能量法结果 |

## 十二、考场上最容易犯的错

1. **阻尼比分母有2**：`ζ = c/(2√(mk))`，不是 `c/√(mk)`
2. **弹簧串联是倒数相加**，并联是直接相加，别搞反
3. **杠杆比要平方**：通过杠杆折算质量/刚度，比例系数要平方
4. **重力不改变固有频率**：坐标移到静平衡位置就行
5. **共振频率不是恰好 ωₙ**：有阻尼时峰值在 `ωₙ√(1-2ζ²)`
6. **隔振起始点是 s=√2**，不是 s=1
7. **脉冲响应里速度初值**：Dirac脉冲让 `ẋ(0⁺)=1/m`，但 `x(0⁺)=0`
8. **模态只定形状不定幅值**：归一化方式可以任选
9. **纯滚动圆盘**：动能 = 平动 ½mv² + 转动 ½Iω²，I = ½mR²（圆柱）
10. **单位统一**：刚度用 N/m、质量用 kg、频率用 rad/s
