## 一、这一章在讲什么？

自由振动是"拉开松手"，受迫振动是**有一个外力一直在推**。

**核心问题**：给一个周期性力 `F(t) = F₀sin(ωt)`，系统的响应是什么？

两个关键发现：
1. 系统**以激励频率 ω 振动**（不是自己的固有频率 ωₙ！）
2. 振幅取决于 ω 离 ωₙ 有多近——**越近越大，重合就是共振**

## 二、谐波激励下的受迫振动（第06讲：核心中的核心）

### 2.1 物理模型

```
     ┌───┐
     │ m │──→ x
     ├─┬─┤
     │k│c│    ← 外力 F(t) = F₀sin(ωt) 直接作用在质量上
     └─┴─┘
```

运动方程：
```
mẍ + cẋ + kx = F₀sin(ωt)
```

标准化：
```
ẍ + 2ζωₙẋ + ωₙ²x = (F₀/m)sin(ωt)
```

### 2.2 全解 = 齐次解 + 特解

```
x(t) = x_h(t) + x_p(t)
       \________/   \______/
        瞬态响应     稳态响应
        （会衰减掉）  （一直存在）
```

- **齐次解 x_h**：和自由振动一样，含 `e^(-ζωₙt)` 因子，有阻尼时会衰减掉
- **特解 x_p**：由激励力决定，以激励频率 ω 振动，不衰减

> 考试只考**稳态响应**（认为瞬态已经衰减完了）

### 2.3 频率比 s —— 核心无量纲参数

```
s = ω/ωₙ
```

这个比值是受迫振动的"开关"——系统行为完全由 s 和 ζ 决定：

| s | 含义 |
|---|------|
| s << 1 | 激励比固有频率慢得多（低频激励） |
| s = 1 | 激励频率 = 固有频率（共振！） |
| s >> 1 | 激励比固有频率快得多（高频激励） |

### 2.4 放大因子 H(s) —— 幅频特性

```
           1
H(s) = ──────────────────
        √((1-s²)² + (2ζs)²)
```

**物理含义**：
```
H(s) = 实际振幅 / 静位移

即：A = (F₀/k) × H(s) = B × H(s)
```

> 静位移 B = F₀/k：如果把 F₀ 慢慢加上去（静态加载），弹簧会变形这么多。  
> H(s) 是"动态放大倍数"。

### 2.5 幅频曲线的三个区

```
H(s)
 ↑
 │        
 │    ··(ζ小)
Q│   ·  · ·              ← 共振峰在 s≈1 处
 │  ·    ·   ··
 │·        ···· ···· → 高频趋近于0
 │ 
1├────────────────────
 │ 所有曲线在 s=0 处汇于 1（= 静位移）
 │
 └───┼────────┼────────→ s=ω/ωₙ
     0        1
  刚度控制  共振区  质量控制
```

**三个区的物理理解**：

| 区间 | H ≈ | 谁主导 | 物理直觉 |
|------|-----|--------|---------|
| **s << 1** | 1 | **刚度 k** | 力变很慢，弹簧来得及反应。响应≈慢悠悠地被推。振幅≈F₀/k |
| **s = 1** | 1/(2ζ) | **阻尼 c** | 力恰好和系统"合拍"。弹簧力和惯性力互相抵消，只剩阻尼力抵抗外力。ζ 越小峰值越恐怖 |
| **s >> 1** | 1/s² → 0 | **质量 m** | 力变得太快，质量惯性跟不上了。力往上拉时质量还没反应过来就被往下推。振幅→0 |

### 2.6 共振

**定义**：激励频率 ω 接近固有频率 ωₙ 时，振幅急剧增大的现象。

**共振时（s=1）**：
```
H(1) = 1/(2ζ) = Q          ← 品质因数就是共振放大倍数

振幅 A_res = F₀/(2kζ) = F₀/(cωₙ)

相位 φ = 90°              ← 位移落后激励力 90°
```

**有阻尼时，真正的幅值最大在**：
```
s_peak = √(1-2ζ²)     ← 略小于1，阻尼越大偏得越多

H_max = ──────────    ← 比 s=1 稍大一点点
         2ζ√(1-ζ²)
```

> 但 ζ 通常很小，所以一般近似认为 s=1 就是共振峰位置。考试看老师习惯。

### 2.7 相频特性 φ(s)

```
        ┌ 2ζs  ┐
φ = arctan│──────│
        └ 1-s² ┘
```

```
φ
180°├─────────────··· → ζ=0（在 s=1 处跳变）
    │           ··
    │         ··   ζ 小
135°│       ··
    │     ··
 90°├────●────          ← 所有曲线在 s=1 处穿过 90°
    │  ··
    │··
 45°│
    │·
  0°├───────────────→ s
    0         1
```

| s 范围 | φ | 物理含义 |
|--------|---|---------|
| s << 1 | ≈ 0° | 位移和力**同相**。力向右推，质量就向右，瞬间跟上 |
| s = 1 | = 90° | 位移**落后力 90°**。力最大时位移为零（速度最大），力为零时位移最大 |
| s >> 1 | ≈ 180° | 位移和力**反相**。力向右推，质量偏偏向左走，完全反着来 |

### 2.8 品质因数 Q

```
Q = 1/(2ζ)

Q = H(s=1)           ← 共振时放大因子 = Q
```

**Q 的物理意义**：
- Q 大（ζ 小）→ 共振峰尖锐 → 阻尼小
- Q 小（ζ 大）→ 共振峰平缓 → 阻尼大
- 类似于钟：Q 大 = 敲一下响很久；Q 小 = 敲一下马上哑

**半功率带宽法测阻尼**（实验方法）：

在幅频曲线上找到 `H = Q/√2`（即功率一半）的两个频率点 ω₁, ω₂：

```
    ωₙ         1
Q = ───── = ─────
    ω₂-ω₁    s₂-s₁
```

## 三、基础激励与隔振（第07讲）

### 3.1 换个激励方式——地板在振

前面是力作用在质量上，现在：**基础/支撑/地板在振动**。

```
        ┌───┐
  x(t)← │ m │    ← 质量的绝对位移
        ├─┬─┤
        │k│c│
   ┌────┴─┴─┴────┐
   │   基础      │ ← x_f(t) = Dsin(ωt)  基础在上下振
   └─────────────┘
```

应用场景：地震时建筑物的晃动、车在不平路面上行驶、精密仪器放在振动台面上。

### 3.2 相对位移 vs 绝对位移

```
x₁ = x - x_f         ← 相对位移（弹簧的变形量 = 质量相对于基础的位移）
x                    ← 绝对位移（质量在空间中的绝对位置）
x_f = Dsin(ωt)       ← 基础位移
```

**运动方程**（用相对位移 x₁ 写）：
```
mẍ₁ + cẋ₁ + kx₁ = mDω²sin(ωt)
```

右边的 `mDω²` 相当于等效激励力幅值——基础振得越快，等效力越大。

### 3.3 相对传递率 η₁

```
           相对位移幅值     s²
η₁(s) = ────────────── = ──────────────────
           基础位移幅值D    √((1-s²)²+(2ζs)²)
```

| s << 1 | s = 1 | s >> 1 |
|--------|-------|--------|
| η₁ → 0 | η₁ → 1/(2ζ) | η₁ → 1 |
| 基础动得慢，弹簧跟着变形小 | 共振，相对位移很大 | 基础动得快，质量根本跟不上，弹簧被大幅拉伸 |

### 3.4 绝对传递率 η₂（隔振设计的关键！）

```
           绝对位移幅值      ┌                ┐½
η₂(s) = ────────────── =   │    1+(2ζs)²     │
           基础位移幅值D     │ ────────────── │
                            └ (1-s²)²+(2ζs)² ┘
```

### 3.5 隔振的核心判断——s = √2

```
η₂
 ↑
 │  ··(ζ大)
 │  ·  ··
 │  ·    ··             ← s=√2 处所有曲线交汇！
>1│·       ··
  │         · ··(ζ小)
  │           ·  ·
 1├────────────●─────── ← 交汇点 s=√2，η₂=1
  │              ··
<1│                ··   ← 隔振区：η₂<1
  │                  ··
  │                    ··
 └┼──────────┼──────────→ s
  0         √2
```

| s 范围 | η₂ | 含义 |
|--------|-----|------|
| s < √2 | η₂ > 1 | **放大区**——振动被放大传到质量上 |
| s = √2 | η₂ = 1 | **临界点**——无论阻尼多大，传递率恒为 1 |
| s > √2 | η₂ < 1 | **隔振区**——传到质量上的振动比基础还小 ✓ |

### 3.6 隔振设计的黄金法则

> 要让隔振器有效，固有频率必须低于激励频率的 1/√2（即约 70%）。

实操步骤：
1. 已知激励频率 ω
2. 选弹簧刚度 k，使 `ωₙ = √(k/m) < ω/√2`
3. 通常取 `s > 3` 获得较好隔振效果

**阻尼的悖论**：
- s < √2 区域：**阻尼大 → 峰值低 → 好事**
- s > √2 区域：**阻尼大 → 传递率大 → 坏事！**
- 所以隔振器不能盲目加阻尼，要在共振保护和高频隔振之间权衡

## 四、不平衡质量激励（第08讲）

### 4.1 另一种激励方式——转子的偏心质量

旋转机械（电机、涡轮机、汽车发动机）中，转子不可能完全平衡，存在偏心质量 m 在距轴心 e 处旋转。

```
等效激励力幅值：F₀ = meω²

不是常数！力随转速平方增大。
```

运动方程：
```
Mẍ + cẋ + kx = meω²sin(ωt)
```

其中 M 是总质量（含偏心质量 m），m 是偏心质量，e 是偏心距。

### 4.2 与普通力激励的关键区别

| | 普通力激励 | 不平衡质量激励 |
|------|----------|-------------|
| 激励力幅 | F₀（常数） | meω²（∝ ω²） |
| s→0 响应 | 静位移 F₀/k | **趋近于 0！** |
| s→∞ 响应 | 趋近于 0 | 趋近于 me/M |
| 低频行为 | 近似静力学 | 力太小，几乎不振 |

> ⚠️ **这是考试容易设坑的地方**——看见受迫振动就套 F₀/k，但如果不平衡激励，s→0 时响应是 0，不是 F₀/k！

### 4.3 无量纲响应

定义无量纲幅值比：
```
  B          s²
─────── = ──────────────────
 me/M    √((1-s²)²+(2ζs)²)
```

这个曲线和基础激励的相对传递率 η₁ 形态一样（都是 s² 分子）。

### 4.4 频响函数（FRF）——频域的"指纹"

对系统 `mẍ + cẋ + kx = F₀e^(iωt)`（用复数表示）：

**位移频响函数 = 动柔度 = 导纳**：
```
H(ω) = X/F₀ = ──────────────
              k - mω² + icω

         1          1
    = ─── · ──────────────
         k   (1-s²) + i(2ζs)
```

**幅值**：
```
|H(ω)| = ─────── · H(s)      ← 1/k 乘以之前的放大因子
            k
```

### 4.5 机械阻抗

阻抗 = 导纳的倒数 = "输入/输出"（力除以响应）：

```
位移阻抗 Z_x = k - mω² + icω          ← 动刚度
速度阻抗 Z_v = Z_x/(iω)
加速度阻抗 Z_a = Z_x/(-ω²)
```

### 4.6 传递力（传到基础上的力）

弹簧力 kx 和阻尼力 cẋ 的矢量和传给基础：

```
传递力幅 F_T = √((kA)² + (cωA)²)

传递率 TR = F_T/F₀ = √(1+(2ζs)²) · H(s)

            ┌                ┐½
            │    1+(2ζs)²    │
    TR  =   │ ────────────── │      ← 和绝对传递率 η₂ 完全一样！
            │ (1-s²)²+(2ζs)² │
            └                ┘
```

## 五、工程中的受迫振动问题（第09讲）

### 5.1 傅里叶级数法——处理任意周期性激励

如果激励不是纯正弦，而是方波/锯齿波/三角波等**周期性但非简谐**的力：

**第一步**：把 F(t) 展开为傅里叶级数（一堆正弦+余弦的和）：
```
               a₀   ∞
F(t) = ─── +  Σ  [aₙcos(nωt) + bₙsin(nωt)]
                2   n=1
```

其中系数：
```
     2  ∫₀ᵀ
aₙ = ── │  F(t)cos(nωt) dt
     T  ⌡₀

     2  ∫₀ᵀ
bₙ = ── │  F(t)sin(nωt) dt
     T  ⌡₀
```

也可以写为：
```
         ∞
F(t) =   Σ  cₙcos(nωt + φₙ)
        n=0
```

其中 `cₙ = √(aₙ²+bₙ²)`，`φₙ = arctan(-bₙ/aₙ)`。

**第二步**：对每个谐波分量分别求解稳态响应：
```
对于第 n 阶谐波 Fₙ = cₙcos(nωt+φₙ)：
频率比 = n·s（第 n 阶谐波频率是 nω）

xₙ(t) = (cₙ/k)·H(ns)·cos(nωt + φₙ - ψₙ)

其中 ψₙ = arctan(2ζ·ns/(1-(ns)²))
```

**第三步**：所有谐波响应叠加：
```
x(t) = Σ xₙ(t)
```

### 5.2 物理直觉

- 系统像一个**低通滤波器**
- 基频（n=1）响应最大，高次谐波被系统滤掉
- 如果某个谐波的 `nω ≈ ωₙ`，该谐波会引起共振
- 方波的傅里叶级数只有奇数次谐波（1,3,5...）

### 5.3 吉布斯现象

用有限项傅里叶级数逼近有跳跃的函数时（如方波），在跳跃处出现过冲（约 9% 的 overshoot）。考试知道有这个现象即可。

## 六、受迫振动四种激励形式对比

| 类型 | 激励源 | 等效激励力幅 | 振幅公式特征 | PPT |
|------|--------|------------|------------|-----|

| s | H(s) | η₂(s) | φ(s) | 物理意义 |
|---|------|-------|------|---------|
| 0 | 1 | 1 | 0° | 准静态，所有曲线汇合 |

| → ∞ | 1/s² → 0 | 1/s → 0 | 180° | 惯性跟不上 |

## 十、最容易犯的错

1. **频率比 s 是 ω/ωₙ**，不是 ωₙ/ω，分子分母别搞反
2. **s=1 时 φ=90°**，s>>1 时 φ→180°，s<<1 时 φ→0°
3. **不平衡激励 s→0 时响应→0**，不是静位移！
4. **隔振起点是 s=√2**，不是 s=1（共振点和隔振临界点是两回事）
5. **基础激励中绝对传递率和力传递率公式一样**——同一个 TR 公式
6. **傅里叶级数叠加时要带相位**，不能直接加振幅
7. **H(s) 的分母**是 `√((1-s²)²+(2ζs)²)`，里面是减号，别写成加号
8. **有阻尼真实共振频率**是 `ωₙ√(1-2ζ²)`，比 ωₙ 略小
