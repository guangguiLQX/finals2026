# 阻尼自由振动 · 概念笔记

> PPT 第05讲 · 单自由度系统自由振动（阻尼）  
> 考试必考：ζ 计算、三种阻尼判断、欠阻尼响应、对数衰减率

---

## 一、加了阻尼之后，和前面有什么不同？

**前面（第03-04讲）**：无阻尼，振幅永远不变。理想情况，现实中不存在。

**现在（第05讲）**：加阻尼器 c，振幅越来越小，最终停下来。逼近真实世界。

```
     ┌───┐
     │ m │──→ x      ← 坐标原点取静平衡位置
     ├─┬─┤
     │k│c│            ← 阻尼器和弹簧并联（标准配置）
     └─┴─┘
   固定端
```

运动方程：
```
mẍ + cẋ + kx = 0

标准化（两边除以 m）：
ẍ + 2ζωₙẋ + ωₙ²x = 0

其中 ωₙ = √(k/m)，ζ = c/(2√(mk))
```

---

## 二、阻尼比 ζ —— 整个章节的核心

### 2.1 定义

```
     c        c        c
ζ = ─── = ────── = ──────
     c_c    2√(mk)   2mωₙ
```

- **c_c** = 临界阻尼系数 = `2√(mk) = 2mωₙ`
- ζ 是**无量纲数**，描述阻尼"够不够强"
- ζ 的值决定了系统的一切行为

### 2.2 ζ 的物理直觉

| ζ | 含义 | 日常类比 |
|---|------|---------|
| ζ = 0 | 无阻尼，永不停止 | 理想真空中的弹簧 |
| 0 < ζ < 0.2 | 轻阻尼，振荡多次才停 | 音叉、铃声 |
| 0.2 < ζ < 1 | 中等阻尼，振荡几次就停 | 汽车悬架 |
| ζ = 1 | 临界阻尼，刚好不振荡，最快回位 | 精密仪器减振目标 |
| ζ > 1 | 过阻尼，缓慢爬回，不振荡 | 门上的阻尼闭合器 |

### 2.3 ζ 从哪里来？（两个来源）

**给定物理参数 c, m, k**：
```
ζ = c/(2√(mk))
```

**从实验振幅数据反推**（对数衰减率法）：
```
ζ = δ/(2π)   （当 ζ < 0.2 时近似成立）
```

## 三、特征方程——决定系统命运

设解为 `x = e^(st)`，代入 `ẍ + 2ζωₙẋ + ωₙ²x = 0`：

```
s² + 2ζωₙs + ωₙ² = 0

s₁,₂ = -ζωₙ ± ωₙ√(ζ²-1)
```

**根号里的东西决定一切**：

```
ζ²-1 < 0  →  共轭复根  →  振荡衰减（欠阻尼）
ζ²-1 = 0  →  重实根    →  临界衰减（临界阻尼）
ζ²-1 > 0  →  两个不等实根 → 缓慢衰减（过阻尼）
```

## 四、欠阻尼（ζ < 1）——考试重点

工程中最常见，也是考试唯一会出计算题的情况。

### 4.1 特征根和固有频率

```
s₁,₂ = -ζωₙ ± iω_d

ω_d = ωₙ√(1-ζ²)        ← 阻尼固有频率
```

| ζ | ω_d/ωₙ | 频率变化 |
|---|---------|---------|
| 0.01 | 0.99995 | 几乎不变 |
| 0.05 | 0.9987 | -0.13% |
| 0.10 | 0.9950 | -0.5% |
| 0.20 | 0.9798 | -2% |
| 0.50 | 0.8660 | -13% |

> 对大多数工程系统 ζ < 0.2，ω_d ≈ ωₙ，频率变化可忽略。  
> **但振幅衰减不能忽略！**

### 4.2 通解（考试必须背下来）

```
x(t) = e^(-ζωₙt) · [C₁cos(ω_d t) + C₂sin(ω_d t)]
```

代入初始条件 x(0) = x₀, ẋ(0) = v₀：

```
x(t) = e^(-ζωₙt) · [x₀cos(ω_d t) + ────────·sin(ω_d t)]
                                       ω_d

其中 v₀ + ζωₙx₀ = ẋ(0) + ζωₙ·x(0)
```

### 4.3 响应图像的解读

```
x(t)
 ↑
A │···                              ← 上包络线：+Ae^(-ζωₙt)
  │    ··    ··    ··    
  │  ·   ···    ····   ··           ← 包络线内以 ω_d 振荡
 0├──┼────┼────┼────┼────→ t
  │      ··    ··    ··
-A │         ···                              ← 下包络线：-Ae^(-ζωₙt)
  │
  └── T_d = 2π/ω_d ─┘

T_d > T（无阻尼周期），但一般差异极小
```

**三个特征**：
1. **振荡频率** ω_d（≈ ωₙ）
2. **振幅指数衰减**，衰减速率由 `ζωₙ` 决定
3. **包络线是 ±A·e^(-ζωₙt)**，每过一个周期，振幅缩小到 `e^(-δ)` 倍

### 4.4 容易写错的项

```
分子是  v₀ + ζωₙx₀    ← 别忘了 ζωₙx₀ 这一项！
分母是  ω_d           ← 是 ω_d，不是 ωₙ！

x(t) = e^(-ζωₙt)[...]  ← 指数里的系数是 ζωₙ
                                                    ↑ 不是 ω_d
```

## 五、临界阻尼（ζ = 1）

```
s₁,₂ = -ωₙ（重根）

x(t) = e^(-ωₙt) · (C₁ + C₂t)
     = e^(-ωₙt) · [x₀ + (v₀+ωₙx₀)·t]
```

**特点**：不振荡，以**最快速度**回到平衡位置。

这是很多减振器设计的目标——刚好不振荡，回位最快。

---

## 六、过阻尼（ζ > 1）

```
s₁,₂ = -ζωₙ ± p    其中 p = ωₙ√(ζ²-1)

x(t) = e^(-ζωₙt) · [C₁cosh(pt) + C₂sinh(pt)]
     = e^(-ζωₙt) · [x₀cosh(pt) + ────────·sinh(pt)]
                                      p
```

**一个反直觉的事实**：阻尼越大（ζ >> 1），回位**反而越慢**！

原因：阻尼太大，质量块被"黏住"了，移动极其缓慢。

```
x(t)
 ↑
x₀├···
  │  ··
  │    ··     ← ζ稍微大于1
  │      ···
  │         ····   ← ζ很大（回位很慢！）
 0├──────────────→ t
```

## 七、对数衰减率 δ ——连接理论和实验

### 7.1 定义

```
      ┌  x(tᵢ)  ┐
δ = ln│──────────│
      └ x(tᵢ+T_d)┘
```

即：**相邻两个振幅峰值的比值，取自然对数**。

### 7.2 δ 和 ζ 的关系

```
         2πζ
δ = ──────────
      √(1-ζ²)
```

**小阻尼近似**（ζ < 0.2）：

```
δ ≈ 2πζ
ζ ≈ δ/(2π)
```

> 这是**实验测量阻尼比的标准方法**——  
> 1. 给系统一个初始扰动  
> 2. 用传感器记录衰减振动  
> 3. 从波形上测量相邻峰值的比值  
> 4. 算出 δ，再算出 ζ

### 7.3 跨 j 个周期的测量

实际测量中，一个周期的衰减太小不好测，可以跨 j 个周期：

```
δ = (1/j) · ln(x(tᵢ) / x(tᵢ + j·T_d))
```

**为什么这样更准？**
- 测量误差被 j 稀释（除以了 j）
- 幅度差更大，信噪比更好

### 7.4 数值示例

已知某系统，第 1 个峰值 x₁ = 10mm，第 10 个峰值 x₁₀ = 1mm，跨 9 个周期：

```
δ = (1/9) · ln(10/1) = (1/9) · ln(10) = 0.256

ζ = δ/(2π) = 0.256/(2π) = 0.041  （0.2以下，近似有效）
精确值：ζ = δ/√(4π²+δ²) = 0.256/√(4π²+0.256²) = 0.041  （和近似几乎一致）
```

## 八、三种阻尼情况完整对比

| | 欠阻尼 ζ<1 | 临界阻尼 ζ=1 | 过阻尼 ζ>1 |
|---|-----------|------------|----------|
| 特征根 | 共轭复根 | 重实根 | 两个不等实根 |
| 振荡？ | 是 | 否 | 否 |
| 回位速度 | 中（振荡着衰减） | **最快**（一滑到位） | 慢（黏住了） |
| 方程解 | `e^(-ζωₙt)[···cos+···sin]` | `e^(-ωₙt)(C₁+C₂t)` | `e^(-ζωₙt)[···cosh+···sinh]` |
| 关键频率 | ω_d = ωₙ√(1-ζ²) | — | — |
| 考试频率 | ⭐⭐⭐⭐⭐ | ⭐（概念题） | ⭐（概念题） |

## 九、从物理参数到响应的完整流程

```
已知：m, k, c, x₀, v₀

Step 1：ωₙ = √(k/m)

Step 2：c_c = 2√(mk) = 2mωₙ

Step 3：ζ = c/c_c = c/(2mωₙ)     ← 也即 ζ = c/(2√(mk))

Step 4：判断
  ζ < 1 → 欠阻尼（考试主力）
  ζ = 1 → 临界阻尼
  ζ > 1 → 过阻尼

Step 5（欠阻尼）：
  ω_d = ωₙ√(1-ζ²)
  x(t) = e^(-ζωₙt)[x₀cos(ω_d t) + (v₀+ζωₙx₀)/ω_d · sin(ω_d t)]

Step 6（如果需要）：
  δ = 2πζ/√(1-ζ²) ≈ 2πζ
```

## 十、阻尼对不同物理量的影响

| 物理量 | 无阻尼 | 有阻尼（ζ<1） | 影响程度 |
|--------|--------|-------------|---------|
| 固有频率 | ωₙ | ω_d = ωₙ√(1-ζ²) | 较小（ζ=0.2时仅降2%） |
| 周期 | T = 2π/ωₙ | T_d = 2π/ω_d | 较小（ζ=0.2时仅增2%） |
| 振幅 | 不衰减 | 每周期× e^(-δ) | **极大！** ζ=0.2时每周期剩28% |
| 相位 | φ 恒定 | φ 恒定 | 无影响 |

> 核心结论：**阻尼对频率影响小，对振幅影响大。**  
> 所以很多时候近似用 ωₙ 代替 ω_d 也行，但振幅衰减必须严格算。

## 十一、PPT 里例题透露的考点

### 例题类型1：直接给定参数求响应
```
给 m, k, c, x₀, v₀ → 算 ζ → 判断类型 → 写 x(t)
注意：v₀ 可能是 0（只拉到某个位置松手），
      也可能是非零（给了一个初始速度）
```

### 例题类型2：对数衰减率反求阻尼比
```
给连续振幅数据 → 算 δ → 算 ζ
考试可能给 j 个周期的数据来测试 δ = (1/j)·ln(xᵢ/xᵢ₊ⱼ)
```

### 例题类型3：配合能量法
```
先用能量法求 ωₙ（第04讲内容）→ 再根据给的 c 算 ζ → 写响应
```

## 十二、最容易犯的五个错

### 1. 阻尼比分母忘了系数 2
```
❌ ζ = c/√(mk)
✅ ζ = c/(2√(mk))

❌ ζ = c/(mωₙ)
✅ ζ = c/(2mωₙ)
```

### 2. ω_d 的根号里搞错方向
```
❌ ω_d = ωₙ√(ζ²-1)     ← 这是过阻尼用的 p
✅ ω_d = ωₙ√(1-ζ²)     ← 欠阻尼的阻尼固有频率
```

### 3. 响应公式分子漏 ζωₙx₀
```
❌ x(t) = e^(-ζωₙt)[x₀cos(ω_d t) + (v₀/ω_d)sin(ω_d t)]
✅ x(t) = e^(-ζωₙt)[x₀cos(ω_d t) + (v₀+ζωₙx₀)/ω_d·sin(ω_d t)]
                                       ↑↑↑↑↑↑↑ 这一项！
```
物理原因：即使 v₀=0，阻尼力也会在初始时刻产生等效的速度效应。

### 4. 对数衰减率公式用错
```
❌ δ = ln(xᵢ - xᵢ₊₁)        ← 是比值再取 ln，不是差
✅ δ = ln(xᵢ/xᵢ₊₁)
```

### 5. e 指数写成 ω_d
```
❌ x(t) = e^(-ω_d t)[...]    ← 指数里是 ζωₙ
✅ x(t) = e^(-ζωₙt)[...]
```
包络线的衰减速率由 ζωₙ 决定，不是 ω_d。ω_d 只是振荡频率。
