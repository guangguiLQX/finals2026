# 任意激励下的响应 · 概念笔记

> PPT 第10讲 · 单自由度系统任意激励下的响应（时域方法）  
> 谐波激励用频域法（第06-09讲），非谐波激励用时域法（本章）

---

## 一、这一章解决什么问题？

前面第06-09讲都在处理**简谐激励**（正弦/余弦力），因为：
- 简谐激励最基础
- 傅里叶分解能把任何周期激励变成简谐激励之和

但是！如果激励是**非周期、非简谐**的——比如：
- 一次冲击（锤击、碰撞）
- 矩形脉冲（突加力→持续→突然卸载）
- 地震加速度记录（随机波形）

**谐波法不管用了，需要时域方法。**

两大工具：
1. **单位脉冲响应 h(t)** —— 系统在时域的"指纹"
2. **Duhamel 积分**（卷积积分）—— 任意激励 = 无数小脉冲的叠加

---

## 二、Dirac δ 函数（单位脉冲）

### 2.1 是什么？

一个理想化的**瞬时冲击**：
```
δ(t-τ) = 0    当 t ≠ τ
∫_{-∞}^{∞} δ(t-τ) dt = 1
```

**物理理解**：
```
     ↑ 高度 = ∞（无穷大）
     │
     │  ┃ ← 宽度 → 0，面积 = 1（单位冲量）
     │  ┃
─────┼──┸────→ t
     │  τ
```

不要把它当普通函数——它是一个"广义函数/分布"。数学上定义为宽度 ε→0、高度 1/ε→-∞ 的矩形脉冲的极限。

### 2.2 核心性质：筛选性

```
∫_{-∞}^{∞} f(t)·δ(t-τ) dt = f(τ)
```

δ 函数把 f(t) 在 t=τ 处的值"筛选"出来。这个性质是推导 Duhamel 积分的关键。

## 三、单位脉冲响应 h(t) —— 系统的时域指纹

### 3.1 是什么？

在 t=0 给系统一个**单位冲量**（δ(t)），初始条件为零，系统的响应叫 h(t)。

运动方程：
```
mḧ + cḣ + kh = δ(t),    h(0⁻)=0, ḣ(0⁻)=0
```

### 3.2 脉冲的作用效果——速度突变

对运动方程从 t=0⁻ 到 t=0⁺ 积分：
```
∫ mḧ dt + ∫ cḣ dt + ∫ kh dt = ∫ δ(t) dt = 1

位移项和阻尼项积分 → 0（因为 h 和 ḣ 有限）
惯性项：m[ḣ(0⁺) - ḣ(0⁻)] = 1

→ ḣ(0⁺) = 1/m      ← 速度突变！
→ h(0⁺) = 0          ← 位移连续（来不及动）
```

> ⚠️ **关键理解**：脉冲在一瞬间给了质量一个初速度 1/m，但来位移还来不及变化。脉冲过后就是**以这个初速度开始的自由振动**！

### 3.3 h(t) 的公式

**欠阻尼（ζ < 1）**：
```
        1
h(t) = ──── · e^(-ζωₙt) · sin(ω_d t)      t ≥ 0
       mω_d
```

**无阻尼（ζ = 0）**：
```
        1
h(t) = ──── · sin(ωₙt)                      t ≥ 0
       mωₙ
```

### 3.4 h(t) 的物理意义

```
h(t)
 ↑
 │  ··                                   ← 初始速度 1/m 启动的自由振动
 │  ·  ··    ··                          ← 如果有阻尼，振幅逐渐衰减
 │··    ····   ····→ t
 │  ··  
 │    ··  ··  ··
```

| h(t) 的特征 | 含义 |
|------------|------|
| 初始斜率 = 1/m | 脉冲给的速度 |
| 振荡频率 = ω_d | 系统的阻尼固有频率 |
| 衰减速率 = ζωₙ | 系统的阻尼特性 |
| **完全由 m, k, c 决定** | **和激励无关！是系统本身的属性** |

### 3.5 h(t) 和 H(ω) 的关系

```
h(t) ←→ H(ω)   是一对傅里叶变换对

H(ω) = ℱ{h(t)}   频响函数是脉冲响应的傅里叶变换
h(t) = ℱ⁻¹{H(ω)}  脉冲响应是频响函数的逆傅里叶变换
```

时域和频域是描述同一个系统的两种视角，信息完全等价。

## 四、Duhamel 积分（卷积积分）

### 4.1 核心思想：切成无穷多小脉冲

```
任意力 F(t)：
                                           
F(t)↑        ··                           
    │      ··  ··                           
    │    ··      ··                         
    │  ··          ··                       
    │··              ··                    
    └┼──┼──┼──┼──┼──┼──→ t               
     τ₁ τ₂ τ₃                              
                                           
在 τ 时刻，一小段 F(τ)dτ 视为一个冲量 = F(τ)dτ 的脉冲
这个脉冲产生的响应 = F(τ)dτ · h(t-τ)
所有脉冲的响应加起来 → 积分
```

### 4.2 公式

```
x(t) = ∫₀ᵗ F(τ) · h(t-τ) dτ     ← Duhamel 积分（卷积）
```

展开形式（欠阻尼）：
```
       1    ∫₀ᵗ
x(t) = ──── │  F(τ) · e^(-ζωₙ(t-τ)) · sin[ω_d(t-τ)] dτ
      mω_d  ⌡₀
```

加上初始条件：
```
x(t) = e^(-ζωₙt)[x₀cos(ω_d t) + ────────sin(ω_d t)]
                                 ω_d

           1    ∫₀ᵗ
       + ──── │  F(τ)·e^(-ζωₙ(t-τ))·sin[ω_d(t-τ)] dτ
         mω_d  ⌡₀

       \_________________/   \_______________________________________/
          自由振动响应                     受迫振动响应
       （初始条件引起）                 （外力引起，Duhamel积分）
```

### 4.3 四种等价形式

```
x(t) = ∫₀ᵗ F(τ)·h(t-τ) dτ        ← "激励在前，响应在后"
x(t) = ∫₀ᵗ F(t-τ)·h(τ) dτ        ← "积分变量为延迟时间"
x(t) = F(t) ∗ h(t)               ← 卷积记号
x(t) = h(t) ∗ F(t)               ← 卷积可交换
```

## 五、矩形脉冲响应（PPT 重点例题）

### 5.1 问题描述

```
F(t)
 ↑
Q₀├────────┐
  │        │     ← 持续 t₀ 的矩形脉冲
  │        │
 0├────────┼────→ t
  │   t₀   │
```

**阶段1**（0 ≤ t ≤ t₀）：力持续作用  
**阶段2**（t ≥ t₀）：力突然撤去，系统自由振动

### 5.2 无阻尼解

**阶段1（加载段，0 ≤ t ≤ t₀）**：
```
F(τ) = Q₀（常数）
代入 Duhamel 积分或直接解受迫振动：

x(t) = (Q₀/k)(1 - cos ωₙt) = B(1 - cos ωₙt)
```

物理图像：响应围绕静位移 B=Q₀/k 上下振荡，振幅范围 [0, 2B]。

**阶段2（自由振动段，t ≥ t₀）**：
用 Duhamel 积分从 0 积到 t₀（之后力为零）：
```
             Q₀
x(t) = ──── [cos ωₙ(t-t₀) - cos ωₙt]
              k
```

也可以用三角恒等式改写为：
```
        2Q₀
x(t) = ──── · sin(ωₙt₀/2) · sin[ωₙ(t - t₀/2)]
         k
```

### 5.3 关键发现——脉冲时长影响最大响应

改写后的公式：
```
振幅 ∝ sin(ωₙt₀/2)

当 ωₙt₀/2 = π/2  →  t₀ = π/ωₙ = T/2  时，sin=1，振幅最大
当 ωₙt₀/2 = π    →  t₀ = 2π/ωₙ = T    时，sin=0，振幅为零！
```

| t₀ | sin(ωₙt₀/2) | 结果 |
|----|-------------|------|
| T/2 | 1 | **振幅最大**（2倍静位移） |
| T | 0 | 卸载后完全静止！ |
| T/4 | √2/2 | 中等振幅 |

> ⚠️ **半周期脉冲放大最大**——冲击持续时间恰好等于固有周期的一半时，能量输入最有效。

## 六、多段常值力激励（分段法）

### 6.1 问题

力分多段，每段常数：
```
F(t)
 ↑
Q₁├────┐
  │    └────┐
Q₂├─────────┼────┐
  │         └────┘
 0├────┼────┼────→ t
      t₁   t₂
```

### 6.2 方法：力的开关叠加

将多段常值力分解为**阶跃力的加加减减**：

```
在 t₁ 时刻：去掉 Q₁（相当于加 -Q₁），加上 Q₂
在 t₂ 时刻：去掉 Q₂（相当于加 -Q₂）
```

每个阶跃力单独求解，按时间区间叠加。

### 6.3 三段区间的解

**区间1**（0 ≤ t ≤ t₁）：只有 Q₁ 作用
```
x(t) = (Q₁/k)(1 - cos ωₙt)
```

**区间2**（t₁ ≤ t ≤ t₂）：Q₁的卸载效应 + Q₂的加载效应
```
x(t) = (Q₁/k)[cos ωₙ(t-t₁) - cos ωₙt]        ← Q₁卸载
     + (Q₂/k)[1 - cos ωₙ(t-t₁)]              ← Q₂加载
```

**区间3**（t ≥ t₂）：所有力都已卸载，纯自由振动
```
x(t) = (Q₁/k)[cos ωₙ(t-t₁) - cos ωₙt]        ← Q₁卸载效应
     + (Q₂/k)[cos ωₙ(t-t₂) - cos ωₙ(t-t₁)]  ← Q₂卸载效应
```

### 6.4 分段法的核心技巧

> 把每个"常值力的开始"当作正阶跃，每个"常值力的结束"当作等大反号的负阶跃，在时间轴上叠加。

## 七、时域方法 vs 频域方法

| | 时域法（本章） | 频域法（第06-09讲） |
|---|-------------|-----------------|
| 核心函数 | h(t) 脉冲响应 | H(ω) 频响函数 |
| 适用范围 | 任意激励（周期+非周期） | 周期激励、稳态响应 |
| 计算方式 | Duhamel 积分（卷积） | 傅里叶级数 + 叠加 |
| 难度 | 积分可能复杂 | 代数运算 |
| 能求瞬态？ | ✅ 能 | ❌ 只看稳态 |
| 能求稳态？ | ✅ 能（积分后取 t→∞） | ✅ 能 |
| 二者关系 | `h(t) ↔ H(ω)` 傅里叶变换对 |

> 一个系统，两种视角，完全等价。  
> 考试里：谐波激励 → 频域法（简单）；脉冲/阶跃/非周期 → 时域法。

## 八、阶跃响应（常力突然加上后不撤走）

F(t) = Q₀, t ≥ 0（单位阶跃函数）

**无阻尼**：
```
x(t) = (Q₀/k)(1 - cos ωₙt) = B(1 - cos ωₙt)
```
响应在 [0, 2B] 之间振荡，不衰减。

**有阻尼**：
```
            Q₀    ┌              e^(-ζωₙt)                  ┐
x(t) = ──── ·   │ 1 - ──────────── · cos(ω_d t - φ)      │
             k     └              √(1-ζ²)                    ┘

其中 φ = arctan(ζ/√(1-ζ²))
```

稳态（t → ∞）：`x(∞) = Q₀/k = B`（就是静位移，瞬态振荡衰减完后停在 B 处）。

## 九、冲击响应谱

### 9.1 概念

对给定的冲击波形（如矩形脉冲、半正弦脉冲），画出**最大响应**与**脉冲时长/固有周期比**的关系。

```
最大响应 / 静响应
     ↑
  2.0├──·                     ← 矩形脉冲的最大放大接近2
     │    ··
     │      ····
  1.0├────────·······→ t₀/T
     │
    0└────┼────┼────
         0.5   1.0
```

这个图在工程中用于抗冲击设计——知道冲击时长和结构固有周期，就能查出最大响应。

## 十一、易错点

1. **脉冲响应 h(t) 的初始条件**：`h(0⁺)=0, ḣ(0⁺)=1/m`——位移连续，速度突变
2. **Duhamel 积分上限是 t**，不是 ∞（因果系统不响应未来的力）
3. **矩形脉冲阶段2要从0积到 t₀**，不是 t（力在 t₀ 后为零）
4. **分段法各区间要验证边界连续性**——区间交接处位移和速度应相同
5. **阶跃响应稳态值 = Q₀/k**（静位移）——考试可以当验算
6. **t₀ = T/2 时矩形脉冲放大最大**——冲击与结构的时序关系决定响应大小
7. **PPT 标注的重点题 P60 3.1 和 P61 3.11**——教授亲自划的，很可能考
