# 第一章 绪论

## 1.1 材料力学研究什么？

### 承载能力三要素

构件（机器或结构的组成部分）在工作时必须具有足够的承载能力：

| 要素 | 含义 | 反面案例 |
|------|------|---------|
| **强度** | 抵抗断裂破坏的能力 | 钢丝绳断裂 |
| **刚度** | 抵抗弹性变形的能力 | 车床主轴变形过大导致加工精度下降 |
| **稳定性** | 压力下保持原有平衡状态的能力 | 1907年加拿大魁北克大桥因压杆失稳坍塌 |

### 与理论力学的区别

- **理论力学**：研究刚体（不变形体），用静力平衡方程
- **材料力学**：研究变形体，需要考虑变形对平衡的影响

### 基本假设（必背！填空题常考）

1. **连续性假设**：材料无空隙地充满几何空间 → 可以用微积分
2. **均匀性假设**：各处的力学性质完全相同
3. **各向同性假设**：各个方向的力学性质相同（木材、竹材是各向异性的例外）
4. **小变形条件**：变形远小于原始尺寸，可用原始尺寸计算平衡

### 杆件变形的五种基本形式

| 形式 | 外力特点 | 变形特点 |
|------|---------|---------|
| 轴向拉压 | 外力沿轴线 | 伸长或缩短 |
| 剪切 | 大小相等方向相反、作用线很近 | 截面相对错动 |
| 扭转 | 力偶作用面垂直于轴线 | 横截面绕轴线转动 |
| 平面弯曲 | 外力垂直于轴线 | 轴线由直变曲 |
| 组合变形 | 以上多种组合 | 以上多种组合 |

## 1.2 外力、内力与截面法

### 外力分类

- **体积力**：连续分布于体积内（如重力，N/m³）
- **表面力**：作用于接触表面 → 分布力（N/m² 或 N/m）+ 集中力（N）
- **静载荷 / 交变载荷 / 冲击载荷**（按时间变化分类）

### 截面法（求内力的核心方法）

四步口诀：**切 → 取 → 代 → 平**

```
① 切：在所求内力处假想地将构件切开
② 取：任取其中一部分为研究对象
③ 代：将另一部分对研究对象的作用用截面上的内力代替
④ 平：由研究对象的平衡条件确定内力
```

### 内力分量（六个）

- **轴力 F_N**：沿轴线，与截面垂直
- **剪力 F_Sy、F_Sz**：在截面内
- **扭矩 M_x**：绕轴线
- **弯矩 M_y、M_z**：绕截面内两轴

由六个平衡方程求出：
$$\sum F_x = 0,\ \sum F_y = 0,\ \sum F_z = 0$$
$$\sum M_x = 0,\ \sum M_y = 0,\ \sum M_z = 0$$

> ⚠️ **重要**：切开之前不可用力的可传性原理，不可用等效力系代替外力！

## 1.3 应力与应变

### 应力（Stress）

截面上单位面积的内力。围绕 k 点取微面积 ΔA，其上内力 ΔF：

$$p = \lim_{\Delta A \to 0} \frac{\Delta F}{\Delta A} = \frac{dF}{dA}$$

分解为：
- **正应力 σ**：垂直于截面（法向）
- **切应力 τ**：在截面内（切向）

$$p^2 = \sigma^2 + \tau^2$$

**单位**：Pa = N/m²，MPa = 10⁶ Pa，GPa = 10⁹ Pa

### 应变（Strain）

- **正应变（线应变）ε**：单位长度的伸长/缩短量，ε = ΔL/L
- **切应变（角应变）γ**：两棱边之间直角的改变量，单位 rad

# 第二章 轴向拉伸和压缩

## 2.1 基本概念

**受力特点**：外力合力作用线与杆轴线重合
**变形特点**：杆沿轴线方向伸长（拉）或缩短（压）

## 2.2 轴力与轴力图

### 轴力 F_N

用截面法求解。**符号规定**：
- 拉力（背离截面）→ 正（+）
- 压力（指向截面）→ 负（−）

**轴力图**：直观反映轴力沿杆长的变化。

### 横截面上的正应力

**推导逻辑**：实验观察 → 平面假设 → 应力均匀分布 → 公式

**实验现象**：横向线和纵向线变形后仍为平行直线，间距变化。

**平面假设**：变形前的横截面变形后仍为平面，各横截面沿杆轴线作相对平移。

**正应力公式**：

$$\sigma = \frac{F_N}{A}$$

- 拉伸为正（拉应力），压缩为负（压应力）
- 适用条件：轴向拉压杆，除外力作用点附近区域

### 斜截面上的应力

$$\sigma_\alpha = \sigma\cos^2\alpha$$
$$\tau_\alpha = \frac{\sigma}{2}\sin 2\alpha$$

**极值**：
- 最大正应力在横截面（α = 0）：σ_max = σ
- 最大切应力在 45° 斜面：τ_max = σ/2

---

## 2.3 强度条件

### 许用应力

$$[\sigma] = \frac{\sigma_{jx}}{n}$$

- 塑性材料：σ_jx = σ_s（屈服极限）
- 脆性材料：σ_jx = σ_b（强度极限）
- n 为安全系数（> 1）

### 强度条件

$$\sigma_{\max} = \frac{F_{N\max}}{A} \leq [\sigma]$$

**三种应用**：
| 问题类型 | 已知 | 求 | 公式 |
|---------|------|-----|------|
| 强度校核 | F, A, [σ] | 是否安全 | σ_max ≤ [σ]? |
| 截面设计 | F, [σ] | A | A ≥ F_Nmax/[σ] |
| 许用载荷 | A, [σ] | [F] | F_Nmax ≤ [σ]·A |

---

## 2.4 胡克定律与变形计算

### 胡克定律

在弹性范围内（σ ≤ σ_p）：

$$\sigma = E\varepsilon$$

另一种形式：
$$\Delta L = \frac{F_N L}{EA}$$

其中：
- **E**：弹性模量（材料常数，单位与应力相同）
- **EA**：抗拉（压）刚度——EA 越大，变形越小

### 泊松比

$$\mu = \left|\frac{\varepsilon'}{\varepsilon}\right|$$
$$\varepsilon' = -\mu\varepsilon$$

横向应变 ε' 与轴向应变 ε 符号相反（拉长则变细）。

### 变形计算

**等截面杆**：
$$\Delta L = \sum \frac{F_{Ni} L_i}{EA}$$

**变截面/变轴力一般公式**：
$$\Delta L = \int_0^L \frac{F_N(x)dx}{EA(x)}$$

---

## 2.5 材料在拉压时的力学性质

### 低碳钢拉伸的四个阶段

| 阶段 | 特征点 | 含义 |
|------|--------|------|
| ① 弹性阶段 | σ_p（比例极限） | 应力-应变成正比 |
| ② 屈服阶段 | σ_s（屈服极限） | 应力不增应变增，出现滑移线 |
| ③ 强化阶段 | σ_b（强度极限） | 材料重新抵抗变形 |
| ④ 颈缩阶段 | 断裂 | 局部截面急剧缩小 |

### 塑性指标

- **延伸率**：δ = ΔL/L₀ × 100%
  - 塑性材料：δ ≥ 5%（如低碳钢 δ≈30%）
  - 脆性材料：δ < 5%（如铸铁 δ≈0.5%）
- **断面收缩率**：ψ = (A-A₁)/A × 100%

### 名义屈服极限 σ₀.₂

对于无明显屈服阶段的材料（如硬铝、50钢），取产生 0.2% 塑性应变时的应力。

### 铸铁的特点

- 无直线段、无屈服、无颈缩
- 抗压强度 ≈ 抗拉强度的 4~5 倍
- 断口约呈 45° 斜面

---

## 2.6 应力集中

截面急剧变化处应力局部增大的现象。

$$K = \frac{\sigma_{\max}}{\sigma_0}$$

- 对**脆性材料**静强度影响大
- 对**塑性材料**静强度影响不大（应力重分布）
- 对**疲劳强度**影响极大（不论塑性还是脆性）

---

## 2.7 超静定问题

**超静定次数** = 未知力个数 − 独立平衡方程个数

**求解三步法**：
1. **静力平衡方程**（有多少列多少）
2. **变形协调方程**（画变形图，找几何关系）
3. **物理方程**（ΔL = F_N·L/EA）

联立求解。

**温度应力与装配应力**：
- 静定结构中不存在（可以自由变形）
- 超静定结构中存在（变形受到约束）

# 第三章 剪切

## 3.1 剪切的概念

**受力特点**：两侧外力大小相等、方向相反、作用线相距很近
**变形特点**：两力作用线之间的截面产生相对错动

**工程实例**：铆钉、销钉、键、螺栓、榫接

**三种破坏形式**：
1. 连接件剪切破坏
2. 连接件与被连接件挤压破坏
3. 被连接件拉断（截面被孔削弱）

## 3.2 剪切实用计算

名义切应力（假设均匀分布）：

$$\tau = \frac{F_S}{A}$$

**剪切强度条件**：

$$\tau = \frac{F_S}{A} \leq [\tau]$$

## 3.3 挤压实用计算

挤压面为半个圆柱面，实用计算中用**直径投影面**：

$$A_{bs} = d \cdot \delta$$

$$\sigma_{bs} = \frac{F_{bs}}{A_{bs}} \leq [\sigma_{bs}]$$

## 3.4 许用应力参考关系

| 材料 | [τ] | [σ_bs] |
|------|-----|--------|
| 塑性材料 | (0.6~0.8)[σ] | (1.7~2.0)[σ] |
| 脆性材料 | (0.8~1.0)[σ] | (0.9~1.5)[σ] |

# 第四章 扭转

## 4.1 扭转的基本概念

**扭转**：一对大小相等、方向相反的力偶，力偶作用面垂直于杆轴。

**轴**：主要发生扭转变形的杆件。

**外力偶矩**（已知功率 P/kW 和转速 n/r·min⁻¹）：

$$M_e = 9549 \times \frac{P}{n} \quad \text{(N·m)}$$

## 4.2 扭矩与扭矩图

用截面法求扭矩。**符号规定——右手螺旋法则**：
- 扭矩矢量离开截面为正
- 扭矩矢量指向截面为负

## 4.3 薄壁圆筒的扭转

- 横截面上无正应力，只有切应力
- 切应力沿圆周均匀分布

$$\tau = \frac{T}{2\pi r_0^2 t}$$

**剪切胡克定律**：
$$\tau = G\gamma$$

$$G = \frac{E}{2(1+\mu)}$$

其中 G 为剪切弹性模量。

---

## 4.4 等直圆杆扭转应力

### 推导三步走

**① 几何方面——平面假设**：
横截面如同刚性平面一样绕轴线转动。
$$\gamma_\rho = \rho\frac{d\phi}{dx}$$

**② 物理方面——胡克定律**：
$$\tau_\rho = G\gamma_\rho = G\rho\frac{d\phi}{dx}$$

**③ 静力学方面——扭矩合成**：
$$T = \int_A \rho\tau_\rho dA$$

联立得**扭转切应力公式**：
$$\tau_\rho = \frac{T\rho}{I_p}$$

### 最大切应力

$$\tau_{\max} = \frac{TR}{I_p} = \frac{T}{W_p}$$

### 截面几何性质

| 截面类型 | 极惯性矩 I_p | 扭转截面系数 W_p |
|---------|-------------|-----------------|
| 实心圆 d | πd⁴/32 | πd³/16 |
| 空心圆 D,d (α=d/D) | πD⁴(1−α⁴)/32 | πD³(1−α⁴)/16 |

### 切应力互等定理

两相互垂直截面上，垂直于交线的切应力数值相等，方向均指向或背离交线。

**纯剪切应力状态**：只有切应力无正应力的状态。

### 斜截面应力

$$\sigma_\alpha = -\tau\sin 2\alpha$$
$$\tau_\alpha = \tau\cos 2\alpha$$

当 α = 45° 时：σ_max = τ（拉），σ_min = −τ（压）

> **破坏规律**：塑性材料抗剪差→沿横截面破坏；脆性材料抗拉差→沿45°螺旋面破坏

---

## 4.5 扭转强度与刚度条件

### 强度条件

$$\tau_{\max} = \frac{T_{\max}}{W_p} \leq [\tau]$$

### 刚度条件

$$\theta_{\max} = \frac{T_{\max}}{GI_p} \leq [\theta]$$

其中 θ 为单位长度扭转角（rad/m），[θ] 常用 (°)/m。

GI_p 称为**抗扭刚度**。

### 相对扭转角

$$\phi_{AB} = \frac{TL}{GI_p}$$

阶梯杆：$$\phi = \sum \frac{T_i L_i}{G_i I_{pi}}$$

变扭矩：$$\phi = \int_0^L \frac{T(x)}{GI_p}dx$$

---

## 4.6 矩形截面杆自由扭转

- 横截面发生**翘曲**（不再保持平面）
- 角点切应力为零
- 长边中点切应力最大
- 圆形截面抗扭性能优于正方形截面

# 第五章 弯曲内力

## 5.1 弯曲的基本概念

**弯曲**：外力垂直于杆轴线时产生的变形。

**梁（Beam）**：主要发生弯曲变形的杆件。

**平面弯曲**：所有外力作用在梁的纵向对称面内，轴线在此面内弯曲成平面曲线。**这是本章讨论的前提条件。**

### 梁的三种基本类型

| 类型 | 支座形式 | 例子 |
|------|---------|------|
| 简支梁 | 一端固定铰 + 一端可动铰 | 普通桥梁 |
| 悬臂梁 | 一端固定、一端自由 | 阳台挑梁 |
| 外伸梁 | 简支梁一端或两端伸出 | 火车轮轴 |

### 三种支座
1. **固定铰支座**：不能移动，可以转动 → F_Ax、F_Ay
2. **可动铰支座**：可沿轴向移动和转动 → F_Ry
3. **固定端**：不能移动也不能转动 → F_Ax、F_Ay、M_A

### 载荷类型
- 集中力 F（N）
- 分布力（均布载荷）q（N/m）
- 集中力偶 M_e（N·m）

---

## 5.2 剪力和弯矩

### 两种弯曲内力

| 内力 | 符号 | 物理意义 | 单位 |
|------|------|---------|------|
| **剪力** | F_s | 抵抗截面错动的内力 | N 或 kN |
| **弯矩** | M | 抵抗截面转动的内力偶矩 | N·m 或 kN·m |

### 正负号约定（重点！）

**剪力 F_s**——"顺正逆负"：
- 使脱离体有**顺时针**转动趋势 → 正（+）
- 取左段时剪力**向下为正**；取右段时剪力**向上为正**

**弯矩 M**——"凹正凸负"：
- 使梁**上压下拉**（下凹，像笑脸）→ 正（+）
- 使梁**上拉下压**（上凸，像哭脸）→ 负（−）

---

## 5.3 剪力方程、弯矩方程与内力图

### 截面法计算步骤

1. 求支座反力（整体平衡方程）
2. 在所求截面处假想切开
3. 取外力较少的脱离体
4. 列平衡方程 → 求出 F_s 和 M

### 绘制剪力图和弯矩图步骤

1. 求支座反力
2. 根据载荷变化点**分段**
3. 列出各段的 F_s(x) 和 M(x) 方程
4. 判断形状，计算控制点数值
5. 绘图（注意：正弯矩画在 x 轴**下方**，即受拉一侧）

---

## 5.4 M-F_s-q 微分关系（快速作图关键）

$$\frac{dF_s}{dx} = q(x)$$
$$\frac{dM}{dx} = F_s(x)$$
$$\frac{d^2M}{dx^2} = q(x)$$

**记忆**：q 积分→ F_s 积分→ M；M 求导→ F_s 求导→ q

### 利用微分关系判断图形状

| 载荷情况 | 剪力图 | 弯矩图 |
|---------|--------|--------|
| q = 0（无分布载荷） | 水平直线 | 斜直线 |
| q = 常数向下 | 右下斜直线 | 上凸二次抛物线 |
| q = 常数向上 | 右上斜直线 | 下凸二次抛物线 |
| 集中力作用处 | **突变**（突变值=F） | **折角**（斜率改变） |
| 集中力偶作用处 | 无变化 | **突变**（突变值=M_e） |
| F_s = 0 处 | —— | M 取极值 |

### 积分关系

$$F_{sB} - F_{sA} = \int_A^B q(x)dx \quad \text{（剪力差=载荷图面积）}$$
$$M_B - M_A = \int_A^B F_s(x)dx \quad \text{（弯矩差=剪力图面积）}$$

---

## 5.5 叠加原理作弯矩图

**前提**：小变形 + 弹性范围（满足胡克定律）

多个载荷的效果 = 各载荷单独作用效果的代数和。

---

## 5.6 平面刚架与曲杆内力

### 平面刚架
- 刚性连接（刚接）：既传力也传力偶
- 内力分量：F_s、M、F_N（三个分量）
- 弯矩图画在受拉一侧，不标正负号

### 曲杆
- 未受力时轴线即为曲线
- 弯矩符号：使微段曲率增加为正

# 第六章 弯曲应力

## 6.1 纯弯曲与横力弯曲

| 类型 | 截面内力 | 应力 |
|------|---------|------|
| 纯弯曲 | 只有 M，F_s = 0 | 只有 σ |
| 横力弯曲 | 同时有 M 和 F_s | 有 σ 和 τ |

---

## 6.2 纯弯曲正应力公式推导

### 实验观察
- 横向线保持直线但相对转动
- 纵向线变为圆弧线
- 上方缩短（受压），下方伸长（受拉）
- 存在一个长度不变的层——**中性层**

### 两个基本假设

1. **平面假设**：横截面变形后仍为平面，且垂直于变形后的轴线，绕中性轴旋转
2. **纵向纤维无挤压假设**：各纵向纤维处于单向应力状态

> 细长梁（l/h > 5）时这两个假设足够精确。

### 三步推导

**① 几何关系**：$$\varepsilon = \frac{y}{\rho}$$

纵向线应变沿截面高度按线性规律分布。

**② 物理关系**（胡克定律）：$$\sigma = E\varepsilon = \frac{Ey}{\rho}$$

正应力沿截面高度也线性分布。

**③ 静力学关系**：
- 轴力为零 → 中性轴通过形心
- 定义**惯性矩**：$$I_z = \int_A y^2 dA$$
- 得：$$\frac{1}{\rho} = \frac{M}{EI_z}$$

最终得**弯曲正应力公式**：

$$\boxed{\sigma = \frac{My}{I_z}}$$

**最大正应力**：
$$\boxed{\sigma_{\max} = \frac{M}{W_z}}$$

其中 $$W_z = \frac{I_z}{y_{\max}}$$ 为抗弯截面系数。

## 6.3 常用截面几何性质（必须背！）

| 截面 | I_z | W_z |
|------|-----|-----|
| 矩形 b×h | bh³/12 | bh²/6 |
| 圆形 d | πd⁴/64 | πd³/32 |
| 空心圆 D,d | π(D⁴−d⁴)/64 | π(D⁴−d⁴)/(32D) |

**平行移轴定理**：$$I_z = I_{zc} + A d^2$$

---

## 6.4 弯曲正应力强度条件

### 塑性材料（拉压等强度）
$$\sigma_{\max} = \frac{M_{\max}}{W_z} \leq [\sigma]$$

### 脆性材料（拉压不等强度）
$$\sigma_{t,\max} \leq [\sigma_t], \quad \sigma_{c,\max} \leq [\sigma_c]$$

### 三种问题
1. **强度校核**：σ_max ≤ [σ]？
2. **截面设计**：W_z ≥ M_max/[σ]
3. **许用载荷**：M_max ≤ W_z·[σ]

---

## 6.5 弯曲切应力

### 矩形截面

$$\tau = \frac{F_s \cdot S_z^*}{I_z \cdot b}$$

其中 S_z* 是所求点以上部分面积对中性轴的静矩。

最大切应力在中性轴处（y = 0）：

$$\tau_{\max} = \frac{3F_s}{2A} = \frac{3F_s}{2bh}$$

切应力沿高度**抛物线分布**。

### 其他截面

| 截面 | τ_max |
|------|-------|
| 矩形 | 3F_s/(2A) |
| 圆形 | 4F_s/(3A) |
| 工字形 | 腹板：F_s·S_z*/(δ·I_z)，近似 τ_max ≈ F_s/(δ·h₀) |

# 第七章 弯曲变形

## 7.1 挠度和转角

| 量 | 符号 | 定义 | 正方向 |
|-----|------|------|--------|
| **挠度** | y（或 w） | 截面形心在垂直轴线方向的位移 | 向下为正 |
| **转角** | θ | 截面绕中性轴转过的角度 | 顺时针为正 |

$$\theta \approx y' = \frac{dy}{dx}$$

## 7.2 挠曲线近似微分方程

$$\boxed{EIy'' = -M(x)}$$

适用条件：弹性范围、小挠度、细长梁。

## 7.3 积分法求变形

1. 列弯矩方程 M(x)
2. 代入微分方程：$$EIy'' = -M(x)$$
3. 积分一次得转角：$$EI\theta = EIy' = -\int M(x)dx + C_1$$
4. 再积分一次得挠度：$$EIy = -\iint M(x)dx + C_1x + C_2$$
5. 利用边界条件确定积分常数 C_1、C_2

### 边界条件

| 支座 | 条件 |
|------|------|
| 固定端 | y = 0，θ = 0 |
| 铰支座 | y = 0 |
| 中间铰 | 左右挠度相等 |

### 连续条件
在分段处：左右挠度相等、左右转角相等。

---

## 7.4 叠加法求变形（考试重点）

**原理**：弹性小变形下，各载荷引起的变形可线性叠加。

**步骤**：
1. 将复杂载荷分解为简单载荷
2. 查表获取各简单载荷下的挠度和转角
3. 代数量加（注意符号）

### 常用结果（需要记住）

| 梁类型 | 载荷 | 最大挠度 |
|--------|------|---------|
| 简支梁 | 中点集中力 F | FL³/(48EI) |
| 简支梁 | 满跨均布 q | 5qL⁴/(384EI) |
| 悬臂梁 | 自由端集中力 F | FL³/(3EI) |
| 悬臂梁 | 全长均布 q | qL⁴/(8EI) |

---

## 7.5 刚度条件

$$y_{\max} \leq [y], \quad \theta_{\max} \leq [\theta]$$

## 7.6 简单超静定梁

利用变形协调条件（多余约束处的位移 = 0）建立补充方程，联立求解。

# 第八章 应力状态分析与强度理论

## 8.1 为什么研究应力状态？

**问题**：同一构件同一点在不同方向截面上的应力不同。材料破坏不一定发生在横截面上（铸铁受压沿 45° 断裂）。

**应力状态**：过构件内一点所有方位截面上应力的集合。

### 基本概念

| 概念 | 定义 |
|------|------|
| **单元体** | 在一点处取的微小正六面体，边长 dx, dy, dz |
| **主平面** | 切应力 τ = 0 的截面 |
| **主应力** | 主平面上的正应力，σ₁ ≥ σ₂ ≥ σ₃ |

### 应力状态分类

| 类型 | 特征 | 示例 |
|------|------|------|
| 单向 | 只有一个主应力 ≠ 0 | 简单拉压 |
| 二向（平面）| 两个主应力 ≠ 0 | 扭转、弯曲 |
| 三向（空间）| 三个主应力都不为 0 | 厚壁容器 |

---

## 8.2 平面应力分析——解析法

### 斜截面应力公式

已知 σ_x, σ_y, τ_xy，求与 x 轴成 α 角的斜截面上的应力：

$$\boxed{\sigma_\alpha = \frac{\sigma_x+\sigma_y}{2} + \frac{\sigma_x-\sigma_y}{2}\cos 2\alpha - \tau_{xy}\sin 2\alpha}$$

$$\boxed{\tau_\alpha = \frac{\sigma_x-\sigma_y}{2}\sin 2\alpha + \tau_{xy}\cos 2\alpha}$$

**符号规定**：
- σ：拉为正，压为负
- τ：顺时针转动趋势为正
- α：从 x 轴逆时针转到斜面法线为正

### 主应力公式

$$\boxed{\sigma_{\max,\min} = \frac{\sigma_x+\sigma_y}{2} \pm \sqrt{\left(\frac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2}}$$

### 主平面方位

$$\boxed{\tan 2\alpha_0 = -\frac{2\tau_{xy}}{\sigma_x-\sigma_y}}$$

两个主平面相差 90°。

### 最大切应力

$$\boxed{\tau_{\max} = \sqrt{\left(\frac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2}}$$

最大值所在平面与主平面成 45°。

---

## 8.3 图解法——莫尔应力圆

**应力圆方程**：

$$\left[\sigma_\alpha - \frac{\sigma_x+\sigma_y}{2}\right]^2 + \tau_\alpha^2 = \left(\frac{\sigma_x-\sigma_y}{2}\right)^2 + \tau_{xy}^2$$

- 圆心：C((σ_x+σ_y)/2, 0)
- 半径：R = √[((σ_x−σ_y)/2)² + τ_xy²]

**对应规则**：单元体转过 α，应力圆对应转过 2α，方向相同。

---

## 8.4 三向应力状态

三个主应力 σ₁ ≥ σ₂ ≥ σ₃ 确定三个圆，最大剪应力：

$$\boxed{\tau_{\max} = \frac{\sigma_1 - \sigma_3}{2}}$$

---

## 8.5 广义胡克定律

### 一般形式

$$\varepsilon_x = \frac{1}{E}[\sigma_x - \mu(\sigma_y + \sigma_z)]$$
$$\gamma_{xy} = \frac{\tau_{xy}}{G}$$

其中剪切模量：
$$G = \frac{E}{2(1+\mu)}$$

### 平面应力特例（σ_z = 0）

$$\varepsilon_x = \frac{\sigma_x - \mu\sigma_y}{E}$$
$$\varepsilon_y = \frac{\sigma_y - \mu\sigma_x}{E}$$
$$\varepsilon_z = -\frac{\mu}{E}(\sigma_x + \sigma_y)$$

### 体积应变

$$\theta = \varepsilon_1 + \varepsilon_2 + \varepsilon_3 = \frac{1-2\mu}{E}(\sigma_1+\sigma_2+\sigma_3)$$

---

## 8.6 应变能

**应变能密度**：$$u = \frac{1}{2}\sigma\varepsilon = \frac{\sigma^2}{2E}$$

三向应力下总应变能密度：

$$u = \frac{1}{2E}[\sigma_1^2 + \sigma_2^2 + \sigma_3^2 - 2\mu(\sigma_1\sigma_2 + \sigma_2\sigma_3 + \sigma_3\sigma_1)]$$

分解为：
- **体积改变能密度 u_v**：只改变体积
- **形状改变能密度 u_d**：只改变形状（畸变能）

$$u_d = \frac{1+\mu}{6E}[(\sigma_1-\sigma_2)^2 + (\sigma_2-\sigma_3)^2 + (\sigma_3-\sigma_1)^2]$$

---

## 8.7 四大经典强度理论

**为什么要强度理论？** 复杂应力状态下不能简单地把某个 σ 直接与 [σ] 比较，需要"等效"为相当应力 σ_r。

统一形式：$$\boxed{\sigma_r \leq [\sigma]}$$

### 汇总表

| 理论 | 创立者 | σ_r 公式 | 适用材料 | 失效类型 |
|------|--------|---------|---------|---------|
| 第一 | Rankine | σ₁ | 脆性 | 断裂 |
| 第二 | Saint-Venant | σ₁−μ(σ₂+σ₃) | 脆性 | 断裂 |
| 第三 | Tresca | σ₁−σ₃ | **塑性** | 屈服 |
| 第四 | von Mises | √[(Δσ₁₂²+Δσ₂₃²+Δσ₃₁²)/2] | **塑性** | 屈服 |
| 莫尔 | Mohr | σ₁−([σ_t]/[σ_c])σ₃ | 拉压不等 | 两者 |

### 弯扭组合常用公式（σ_y = 0）

- **第三强度理论**：$$\sigma_{r3} = \sqrt{\sigma^2 + 4\tau^2}$$
- **第四强度理论**：$$\sigma_{r4} = \sqrt{\sigma^2 + 3\tau^2}$$

圆轴弯扭组合（用弯矩 M 和扭矩 T 表达）：
- $$\sigma_{r3} = \frac{\sqrt{M^2 + T^2}}{W}$$
- $$\sigma_{r4} = \frac{\sqrt{M^2 + 0.75T^2}}{W}$$

---

# 第九章 组合变形

## 9.1 叠加原理与解题步骤

**前提**：线弹性 + 小变形

**通用步骤**：
```
外力分析（分解载荷）
→ 内力分析（画各基本变形内力图，判断危险截面）
→ 应力分析（计算危险点各应力分量）
→ 应力状态分析（求主应力）
→ 强度校核（选用合适的强度理论）
```

## 9.2 斜弯曲

外力不沿主轴方向时发生。将力分解为两个主轴方向：

$$M_z = F\cos\varphi \cdot x, \quad M_y = F\sin\varphi \cdot x$$

应力叠加：
$$\sigma = \frac{M_z \cdot y}{I_z} + \frac{M_y \cdot z}{I_y}$$

**中性轴方程**（σ = 0 处）：
$$\frac{\cos\varphi}{I_z}y + \frac{\sin\varphi}{I_y}z = 0$$

## 9.3 拉（压）弯组合

$$\sigma = \frac{F_N}{A} \pm \frac{M_z \cdot y}{I_z}$$

注意：轴力产生的 σ 是均匀的，弯矩产生的 σ 是线性分布的，叠加后中性轴不再通过形心。

## 9.4 偏心拉压

偏心压力可等效为轴心压力 + 两个方向的弯矩：

$$\sigma = -\frac{F}{A} - \frac{F \cdot e_z \cdot z}{I_y} - \frac{F \cdot e_y \cdot y}{I_z}$$

**截面核心**：偏心压力作用在截面核心区域以内时，截面上只有压应力。

## 9.5 弯扭组合

圆轴同时受弯矩 M 和扭矩 T 时，表面点处于平面应力状态：

正应力：$$\sigma = \frac{M}{W}$$（弯曲产生）

切应力：$$\tau = \frac{T}{W_p} = \frac{T}{2W}$$（扭转产生，对圆截面 W_p = 2W）

**强度校核**：
- 第三理论：$$\sigma_{r3} = \frac{\sqrt{M^2 + T^2}}{W} \leq [\sigma]$$
- 第四理论：$$\sigma_{r4} = \frac{\sqrt{M^2 + 0.75T^2}}{W} \leq [\sigma]$$

# 第十章 压杆稳定

## 10.1 稳定性概念

### 三类平衡

| 类型 | 特征 | 类比 |
|------|------|------|
| 稳定平衡 | 微小扰动后能自动恢复原位 | 碗底小球 |
| 临界平衡 | 微小扰动后在任意位置都能平衡 | 平面小球 |
| 不稳定平衡 | 微小扰动后远离原位 | 山顶小球 |

**临界压力 F_cr**：使压杆保持直线平衡形态的最大轴向压力。

### 工程教训
- 1907年加拿大魁北克大桥：受压弦杆失稳坍塌
- 细长压杆弯曲的原因：初始曲率、载荷偏心、材料不均匀

---

## 10.2 欧拉公式

### 推导思路

从挠曲线微分方程出发：
$$EIy'' = -F_{cr}y$$

令 $$k^2 = \frac{F_{cr}}{EI}$$，得：
$$y'' + k^2y = 0$$

通解：y = A sin(kx) + B cos(kx)

利用边界条件得 sin(kL) = 0，即 kL = nπ。

取 n=1（最小临界力）：

$$\boxed{F_{cr} = \frac{\pi^2 EI}{(\mu L)^2}}$$

### 长度系数 μ

| 约束条件 | μ |
|----------|-----|
| 两端铰支 | **1.0** |
| 一端固定一端自由 | **2.0** |
| 两端固定 | **0.5** |
| 一端固定一端铰支 | **0.7** |

---

## 10.3 临界应力与柔度

### 基本概念

**临界应力**：
$$\sigma_{cr} = \frac{F_{cr}}{A} = \frac{\pi^2 E}{\lambda^2}$$

**柔度（Slenderness Ratio）**：
$$\lambda = \frac{\mu L}{i}$$

**惯性半径**：
$$i = \sqrt{\frac{I}{A}}$$

### 三类压杆

| 类型 | 条件 | 临界应力 | 失效模式 |
|------|------|---------|---------|
| **大柔度杆** | λ ≥ λ_p | σ_cr = π²E/λ²（欧拉公式） | 弹性屈曲 |
| **中柔度杆** | λ_s ≤ λ < λ_p | σ_cr = a − b·λ（直线公式） | 弹塑性屈曲 |
| **小柔度杆** | λ < λ_s | σ_cr = σ_s | 强度破坏 |

其中：
- $$\lambda_p = \pi\sqrt{\frac{E}{\sigma_p}}$$（比例极限对应柔度）
- a、b 为材料常数，考试会给出

---

## 10.4 稳定性校核

### 方法一：安全系数法

$$F \leq \frac{F_{cr}}{n_{st}} \quad \text{或} \quad n = \frac{F_{cr}}{F} \geq n_{st}$$

### 方法二：折减系数法（φ 法）

$$\sigma = \frac{F}{A} \leq \varphi[\sigma]$$

其中 φ 为折减系数（φ ≤ 1，查表得）。

---

## 10.5 解题流程

```
① 计算柔度 λ = μL/i
② 比较 λ 与 λ_p（和 λ_s），判断杆件类型
③ 选用对应公式计算 σ_cr
④ 计算 F_cr = σ_cr·A
⑤ 校核：F ≤ F_cr/n_st
```

---

## 10.6 提高压杆稳定性的措施

1. **减小 μ**：改善杆端约束（自由端→固定端）
2. **减小 L**：增加中间支承，减小计算长度
3. **增大 i**：选择 I/A 大的截面形状（空心优于实心）
4. **合理选材**：大柔度杆选高 E 材料，中柔度杆选高强度材料
