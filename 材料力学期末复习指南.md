## 一、考试结构一览

| 题型 | 题量 | 每题分值 | 总分 | 特点 |
|------|------|---------|------|------|
| 选择题 | 2-4题 | 5分 | 10-20分 | 概念辨析，需理解原理 |
| 填空题 | 2-4题 | 5-10分 | 15-20分 | 简单计算 + 定义默写 |
| 计算题 | 4-5题 | 10-20分 | 60-75分 | **计算为主，占分大头** |

> ⚠️ **关键特征**：计算题占 **60%~75%**，纯概念题极少。证明题在近年不出现（2022明确写0分）。所有题目都附带受力图/结构图，读图能力是基本功。

## 二、必考知识点（三套卷都出现）

### 🔴 弯曲强度 — 概率 100%，分值 15-20 分

**出题方式**：
- 给定梁结构（简支梁/外伸梁），绘制剪力图和弯矩图
- 用 `σmax = Mmax·ymax / Iz ≤ [σ]` 校核强度
- 脆性材料需分别校核拉/压强度：`σt,max ≤ [σt]`，`σc,max ≤ [σc]`
- 有时结合截面设计：已知 [σ]，反求截面尺寸

**核心公式**：
```
σ = M·y / Iz         正应力公式
σmax = Mmax / Wz     最大正应力
τ = Fs·Sz* / (b·Iz)  切应力（矩形截面）
τmax = 3Fs/(2A)      矩形截面最大切应力（中性轴处）
```

**常见坑**：
- 移动载荷问题需要先找最不利位置
- T形/工形截面中性轴不在几何中心，拉压应力不对称
- 别忘了同时校核正应力和切应力

---

### 🔴 弯曲变形（叠加法）— 概率 100%，分值 15 分

**出题方式**：
- 给定外伸梁/悬臂梁，用叠加法求指定截面的挠度和转角
- 需要查表 + 分解载荷 + 线性叠加

**核心方法**：
```
EI·y'' = -M(x)         挠曲线近似微分方程
y = Σyi                挠度叠加（弹性小变形）
θ = Σθi                转角叠加
```

**必须记住的查表结果**（高频）：

| 梁类型 | 载荷 | 最大挠度 |
|--------|------|---------|
| 简支梁 | 中点集中力 F | FL³/(48EI) |
| 简支梁 | 全长均布 q | 5qL⁴/(384EI) |
| 悬臂梁 | 自由端集中力 F | FL³/(3EI) |
| 悬臂梁 | 全长均布 q | qL⁴/(8EI) |

---

### 🔴 应力状态分析 + 强度理论 — 概率 100%，分值 10-15 分

**出题方式**：
- 给定单元体应力（或从内力反推应力），用解析法/图解法求主应力
- 用第三或第四强度理论校核

**核心公式**：
```
σmax/min = (σx+σy)/2 ± √[((σx−σy)/2)² + τxy²]      主应力
tan(2α₀) = -2τxy/(σx−σy)                             主平面方位
τmax = √[((σx−σy)/2)² + τxy²]                        最大切应力

第三强度理论（最大切应力）：σeq3 = σ₁ − σ₃ ≤ [σ]
第四强度理论（von Mises）：  σeq4 = √[(σ₁−σ₂)²+(σ₂−σ₃)²+(σ₃−σ₁)²]/2 ≤ [σ]
弯扭组合第四强度：           σeq4 = √(σ² + 3τ²) ≤ [σ]
```

## 三、高频知识点（三套卷出现两次）

### 🟡 压杆稳定 — 分值 5-15 分

**出题方式**：
- 求临界压力/临界载荷（选择题或填空计算）
- 判断属于大/中/小柔度杆，选对公式
- 刚杆+弹性杆组合体系求失稳载荷

**核心公式**：
```
Fcr = π²EI / (μl)²       欧拉公式（仅大柔度杆可用！）
σcr = Fcr/A = π²E / λ²   临界应力
λ = μl / i                柔度
i = √(I/A)                惯性半径
```

**长度系数 μ**：
- 两端铰支 μ=1.0
- 一端固定一端自由 μ=2.0
- 两端固定 μ=0.5
- 一端固定一端铰支 μ=0.7

**三类杆的判断流程**：
```
① 先算 λ = μl/i
② 比较 λ 与 λp（比例极限柔度）
   λ ≥ λp → 大柔度杆 → 欧拉公式 σcr = π²E/λ²
   λs ≤ λ < λp → 中柔度杆 → 直线公式 σcr = a − b·λ
   λ < λs → 小柔度杆 → 按强度 σcr = σs
```

**常见坑**：
- 欧拉公式ε只在弹性范围有效，算完要验证 λ ≥ λp
- 考试会给出经验公式参数（a, b），直接用即可
- 注意区分是求 Fcr（临界力）还是 qcr（临界均布载荷）

---

### 🟡 组合变形 — 分值 5-15 分

**出题方式**：
- 判断题：给一个空间结构图，判断杆的变形类型
- 计算题：弯扭组合的强度校核、偏心拉伸求许用载荷

**变形类型辨识**：
- 斜弯曲：力不沿主轴 → 双向弯曲
- 拉(压)弯组合：σ = FN/A ± Mz·y/Iz ± My·z/Iy
- 弯扭组合：用第三/第四强度理论（√(σ²+4τ²) 或 √(σ²+3τ²)）
- 偏心拉压：σ = F/A ± Fe·y/Iz ± Fe·z/Iy

**解题步骤**：
```
外力分析（分解简化）→ 内力分析（M图和T图）→ 危险截面判断
→ 危险点应力单元分析 → 主应力计算 → 强度理论校核
```

---

### 🟡 拉压静不定 — 分值 10-15 分

**出题方式**：
- 刚性梁 + 弹性杆组合结构，求各杆轴力

**三步法**：
```
① 静力平衡方程（ΣF=0，ΣM=0）
② 变形协调方程（画变形图，找几何关系）
③ 物理方程（ΔL = FN·L/EA）
联立求解
```

**关键技巧**：画变形图用"切线代替圆弧"法简化几何关系。

---

### 🟡 能量法（卡氏第二定理）— 分值 15 分

**出题方式**：
- 用卡氏第二定理求梁上某点的挠度

**核心公式**：
```
δ = ∂U/∂F          位移 = 应变能对外力的偏导
U = ∫ M²(x)dx/(2EI)  弯曲应变能
```

**技巧**：若所求点无外力，附加虚力 F，求导后令 F=0。

## 四、基础知识点（概念题常考）

### 🟢 基本假设（5分填空题常客）
- **四个基本假设**：连续性、均匀性、各向同性、小变形
- **胡克定律适用范围**：比例极限以内（不是屈服极限！）

### 🟢 连接件实用计算（5-10分填空）
- 剪切强度：`τ = Fs/A ≤ [τ]`
- 挤压强度：`σbs = Fbs/(d·δ) ≤ [σbs]`（注意是投影面积）

### 🟢 截面几何性质（计算题必需）
| 截面 | Iz | Wz |
|------|-----|-----|
| 矩形 b×h | bh³/12 | bh²/6 |
| 圆形 d | πd⁴/64 | πd³/32 |
| 空心圆 D,d | π(D⁴−d⁴)/64 | π(D⁴−d⁴)/(32D) |

### 🟢 广义胡克定律
```
εx = [σx − μ(σy+σz)] / E
γxy = τxy / G
G = E / [2(1+μ)]
```

## 五、出题规律总结

| 规律 | 说明 |
|------|------|
| **计算碾压** | 60-75% 是计算题，纯记忆概念不足10% |
| **每章一题** | 拉压/剪切/扭转/弯曲/应力状态/稳定各出一题 |
| **读图门槛** | 所有题目附图，先受力分析再计算 |
| **公式不给** | 默认你背下来基本公式，卷面不给公式 |
| **重考换题** | 重考题不重复正考题，但知识点完全相同 |
| **难度梯度** | 选择填空相对简单，最后1-2道计算题最难 |
