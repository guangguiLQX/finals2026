# 能量法 · 概念笔记

> PPT 第04讲 · 单自由度系统自由振动 Part 2  
> 考试大题**极高频**考点——几乎年年考

---

## 一、为什么需要能量法？

牛顿法的痛点：
```
对每个物体画隔离体 → 分析力方向 → 列方程 → 处理约束 → ...
```
系统一复杂（滑轮、杠杆、多弹簧），牛顿法**极其繁琐**。

能量法的优势：
```
只写两个标量（动能T、势能V）→ 求导或取最大值 → 直接得 ωₙ
不需要分析力方向，不需要处理约束反力。
```

---

## 二、能量法的两个用法

### 用法一：推导运动方程（等价于牛顿法）

从能量守恒出发：
```
T + V = 常数
d(T+V)/dt = 0     →  得到运动方程  m_eq·ẍ + k_eq·x = 0
```

### 用法二：直接求固有频率（更常用！）

对于简谐振动，在平衡位置动能最大，在最大位移处势能最大：
```
T_max = V_max

T_max = ½·M_eq·(ẋ_max)² = ½·M_eq·(ωₙ·x_max)²
V_max = ½·K_eq·x_max²

½·M_eq·(ωₙ·x_max)² = ½·K_eq·x_max²
ωₙ² = K_eq/M_eq
ωₙ = √(K_eq/M_eq)
```

> **用法二是考试的绝对主力**——不需要解微分方程，直接从能量表达式读出等效质量和等效刚度。

## 三、核心操作：找等效质量 M_eq 和等效刚度 K_eq

### 3.1 基本思路

**第一步**：选一个**广义坐标**——通常是某个质量块的位移 x（或某个转轴的转角 θ）。

**第二步**：把系统里**所有部件**的动能都用广义坐标表示：

```
每个平动物体：½mᵢvᵢ²  → 用 ẋ 表示 vᵢ
每个转动物体：½Iᵢωᵢ²  → 用 ẋ 表示 ωᵢ（通过半径、杠杆比等）
```

**第三步**：把系统里**所有弹簧**的势能都用广义坐标表示：

```
每个弹簧：½kᵢ(变形量)²  → 用 x 表示变形量
```

**第四步**：提出系数：

```
T = ½·(一堆东西)·ẋ² = ½·M_eq·ẋ²   →  M_eq = 那一堆
V = ½·(一堆东西)·x² = ½·K_eq·x²   →  K_eq = 那一堆
```

**第五步**：`ωₙ = √(K_eq/M_eq)`

## 四、弹簧串并联（第04讲例题6）

### 4.1 串联弹簧

```
┌──[k₁]──[k₂]──┐
固定端          质量块
```

两个弹簧**首尾相接**，各承受相同的力。

**关键**：受力相同，总变形 = 变形1 + 变形2。

```
力 F → 弹簧1变形 δ₁ = F/k₁，弹簧2变形 δ₂ = F/k₂
总变形 δ = δ₁+δ₂ = F(1/k₁+1/k₂) = F/K_eq

     1      1      1
  ──── = ──── + ────
   K_eq    k₁     k₂

       k₁·k₂
K_eq = ──────
       k₁+k₂
```

> **记法**：串联弹簧像电阻并联——倒数相加。n 个弹簧串联：`1/K_eq = Σ1/kᵢ`

### 4.2 并联弹簧

```
    ┌──[k₁]──┐
    ├──[k₂]──┤
固定端       质量块
    └────────┘
```

两个弹簧**并排**连接，变形量相同。

**关键**：变形相同，总力 = 力1 + 力2。

```
总力 F = k₁x + k₂x = (k₁+k₂)x = K_eq·x

K_eq = k₁ + k₂
```

> **记法**：并联弹簧像电阻串联——直接相加。n 个弹簧并联：`K_eq = Σkᵢ`

### 4.3 串并联混合

看到复杂弹簧网络，从负载端往回看，逐步化简：
```
先合并并联块 → 再合并串联段 → 重复直到只剩一个等效弹簧
```

## 五、常见折算套路（PPT 例题详解）

### 5.1 含重力影响的系统（第04讲例题1）

```
质量 m 挂在弹簧 k 下方，有重力。
坐标 x 从弹簧原长算起（不是静平衡位置！）
```

**如果坐标从弹簧原长算起**，重力会出现在势能表达式中：
```
V = ½kx² - mgx
```
`d(T+V)/dt=0` → `mẍ + kx = mg` → 坐标平移 `y=x-mg/k` → `mÿ+ky=0`

**如果一开始坐标就在静平衡位置**：
```
V = ½kx²       ← 重力项直接消失！
ωₙ = √(k/m)    ← 和没重力一样
```

> 结论：**重力只改变平衡位置，不影响 ωₙ。** 直接选静平衡位置为坐标原点，万事大吉。

---

### 5.2 弹簧质量不可忽略（第04讲例题2）

```
弹簧自身质量 m_t，一端固定一端连质量块 m。
弹簧各点速度不同：固定端速度为0，质量块端速度为 ẋ。
假设速度沿弹簧线性分布。
```

距固定端距离为 y 处，弹簧微元 dy 的速度 = ẋ·(y/L)，质量 = m_t·dy/L。

动能积分：
```
T_spring = ∫₀ᴸ ½·(m_t/L)·(ẋ·y/L)²·dy = ½·(m_t/3)·ẋ²
```

**总动能**：
```
T = ½mẋ² + ½·(m_t/3)·ẋ² = ½·(m + m_t/3)·ẋ²

M_eq = m + m_t/3          ← ⚠️ 是 /3，不是 /2！
```

---

### 5.3 杠杆系统（第04讲例题3）

```
        弹簧 k₁         弹簧 k₂
          │               │
    ──────┼───────────────┼──────  ← 杠杆
    │  l₁    │    l₂    │  l₃  │
质量 m₁ ←──┼───────────┼──→ 质量 m₂
            O（支点）
```

**关键**：杠杆上各点位移与到支点的距离成正比。

设质量 m₁ 处的位移为 x（广义坐标），则：
```
质量 m₂ 处的位移 = x·(l₂/l₁)
弹簧 k₂ 处的位移 = x·(l₃/l₁)
```

**动能**：
```
T = ½m₁ẋ² + ½m₂(ẋ·l₂/l₁)² = ½[m₁ + m₂(l₂/l₁)²]·ẋ²

M_eq = m₁ + m₂·(l₂/l₁)²
```

**势能**：
```
V = ½k₁x² + ½k₂(x·l₃/l₁)² = ½[k₁ + k₂(l₃/l₁)²]·x²

K_eq = k₁ + k₂·(l₃/l₁)²
```

**固有频率**：
```
       ┌                  ┐½
       │ k₁ + k₂(l₃/l₁)²  │
ωₙ =   │ ──────────────── │
       │ m₁ + m₂(l₂/l₁)²  │
       └                  ┘
```

> ⚠️ **杠杆比的平方**：通过杠杆折算的质量/刚度，比例系数都要平方。这是直接用能量法自动得到的结果。

---

### 5.4 纯滚动圆盘-弹簧系统（第04讲例题4）

```
圆盘质量 m，半径 R，纯滚动（不打滑）
弹簧 k₁ 连在距离圆心 a 处，弹簧 k₂ 连在距离圆心 b 处（另一侧）
```

圆盘做纯滚动时：
```
圆心位移 = x
圆盘转角 θ = x/R  （不打滑条件）
角速度 θ̇ = ẋ/R
```

**动能 = 平动 + 转动**：
```
T = ½mẋ² + ½I_c·θ̇² = ½mẋ² + ½·(½mR²)·(ẋ/R)²
  = ½mẋ² + ¼mẋ² = ½·(3m/2)·ẋ²

M_eq = 3m/2
```

> 对于匀质圆柱/圆盘：`I_c = ½mR²`  
> 对于匀质圆环：`I_c = mR²`，则 `M_eq = 2m`  
> 对于匀质球：`I_c = ⅖mR²`，则 `M_eq = 7m/5`

**势能**（两个弹簧变形量不同——因为连接点在圆盘上不同位置）：
```
弹簧 k₁ 连接点距圆心 a，当圆盘转 θ 时该点水平位移 = x + aθ = x + a·x/R = x(1+a/R)
弹簧 k₂ 连接点距圆心 b（另一侧），水平位移 = x - b·x/R = x(1-b/R)

V = ½k₁[x(1+a/R)]² + ½k₂[x(1-b/R)]²
  = ½[k₁(1+a/R)² + k₂(1-b/R)²]·x²

K_eq = k₁(1+a/R)² + k₂(1-b/R)²
```

### 5.5 滑轮-绳索-质量系统（第04讲例题5）

```
质量 m 悬挂在绳索一端
绳索绕过滑轮（质量 M，半径 R，转动惯量 I = ½MR²）
绳索另一端连弹簧 k₂
质量块上还直接连着弹簧 k₁
```

设质量 m 的位移为 x（向下为正）：

**位移关系**（绳索不可伸长）：
```
质量下降 x → 滑轮转 x/R → 弹簧 k₂ 缩短 x
弹簧 k₁ 也伸长 x
```

**动能**：
```
质量：½mẋ²
滑轮：½I·(ẋ/R)² = ½·(½MR²)·(ẋ/R)² = ¼Mẋ²

T = ½mẋ² + ¼Mẋ² = ½(m + M/2)ẋ²

M_eq = m + M/2
```

**势能**：
```
弹簧 k₁：½k₁x²
弹簧 k₂：½k₂x²

V = ½(k₁+k₂)x²

K_eq = k₁ + k₂
```

（重力势能抵消——因为平衡位置已扣除）

**固有频率**：
```
ωₙ = √((k₁+k₂)/(m+M/2))
```

## 六、能量法解题标准步骤（考试答题模板）

> 这是你考试大题的框架，直接按这个来：

### Step 1：选广义坐标，画草图标注
```
"以质量 m 的位移 x 为广义坐标"
```

### Step 2：建立运动关系
```
用 x 表示各部件速度、各弹簧变形量
（注意杠杆比、滑轮半径、纯滚动条件等）
```

### Step 3：写动能 T
```
T = ½m₁v₁² + ½I₁ω₁² + ... 
  = ½·M_eq·ẋ²
```

### Step 4：写势能 V
```
V = ½k₁(变形₁)² + ½k₂(变形₂)² + ...
  = ½·K_eq·x²
```

### Step 5：利用 T_max = V_max
```
设 x(t) = A·sin(ωₙt + φ)
则 ẋ_max = ωₙ·A
T_max = ½M_eq·(ωₙA)²
V_max = ½K_eq·A²
令 T_max = V_max → ωₙ² = K_eq/M_eq
```

### Step 6：写出结果
```
ωₙ = √(K_eq/M_eq)  [rad/s]
fₙ = ωₙ/(2π)       [Hz]
T = 2π/ωₙ           [s]
```

## 七、绕晕你的三个易错点

### 7.1 "平方"来源
```
速度 v 和位移 x 线性相关（v = ẋ × 某个比例）
→ 动能里比例系数自动平方

例如杠杆比 l₂/l₁：
位移关系：x₂ = x₁·(l₂/l₁)
速度关系：v₂ = v₁·(l₂/l₁)
动能：½m₂v₂² = ½m₂(l₂/l₁)²·v₁²
              ↑ 自动平方了
```

### 7.2 纯滚动不要漏转动动能
```
纯滚动圆盘：
T = ½mv² + ½I_c·(v/R)²
  平动    转动
两者量级相当，不能只算平动！
```

### 7.3 弹簧变形量的符号不重要（平方后都正）
```
弹簧变形量 = x_object - x_spring_base
不管正负，½k(变形)² = ½kx²，平方消掉了符号
```

## 八、能量法 vs 牛顿法 选择指南

| 情况 | 推荐方法 |
|------|---------|
| 简单弹簧-质量（1-2个弹簧） | 牛顿法更快 |
| 有滑轮、杠杆、多弹簧 | **能量法**——不用管约束力 |
| 纯滚动、复合运动 | **能量法**——平转耦合自动处理 |
| 弹簧串并联网络 | 先化简为等效弹簧，再用任何方法 |
| 要求写运动方程 | 能量法 d(T+V)/dt=0 或牛顿法都可以 |

---

## 九、考试中能量法的变体

1. **纯弹簧串并联**（送分题）——正确判断串并联关系
2. **杠杆/滑轮折算**（中档题）——注意比例系数的平方
3. **纯滚动圆盘**（中档题）——平动+转动动能都要写到
4. **多弹簧多质量混合**（难题）——选好广义坐标，系统化地写 T 和 V
5. **加阻尼再问对数衰减率**（综合题）——能量法求 ωₙ 只是第一步，后面接阻尼分析

能量法的结果 ωₙ 往往只是大题第一问，后面可能接：
- 求阻尼比 ζ
- 求受迫振动响应
- 求隔振传递率

所以**能量法必须又快又准**——它是后续计算的基础。
