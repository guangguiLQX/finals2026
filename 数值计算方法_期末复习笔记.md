# <a id="ch1"></a>第1章 引论（误差分析）

## 1.1 数值计算是什么

> **定义：** 研究求解数学问题**近似解**的方法、过程及其理论的数学分支。

**解决实际问题的流程：**

```
实际工程问题 → 数学力学模型 → 数值计算方法 → 程序设计 → 数值结果 → 分析
```

---

## 1.2 四种误差来源（⭐必考）

| 误差类型 | 产生原因 | 是否研究 |
|---------|---------|:---:|
| **模型误差** | 数学模型是对实际问题的近似，如铝棒长度公式 $l_t = L_0(1+at)$ | ❌ |
| **观测误差** | 测量手段限制，如温度、长度测量不准 | ❌ |
| **截断误差（方法误差）** | 用近似方法代替精确解，如用 Taylor 多项式近似函数 | ✅ |
| **舍入误差** | 计算机字长有限，如 $\pi \approx 3.14159$ | ✅ |

> 🎯 **本课程主要研究：截断误差 + 舍入误差**

**截断误差例子：** $f(x) = p_n(x) + R_n(x)$，余项 $R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}x^{n+1}$

---

## 1.3 绝对误差与相对误差

### 绝对误差

$$e(x^*) = x - x^*$$

（$x$ 为精确值，$x^*$ 为近似值）

**绝对误差限 $\varepsilon^*$：** 满足 $|x - x^*| \leq \varepsilon^*$

此时精确值的范围：$x^* - \varepsilon^* \leq x \leq x^* + \varepsilon^*$，记作 $x = x^* \pm \varepsilon^*$

> 🌰 **例题：** 电子秤读数 $x^* = 4500\text{g}$，误差不超过 $10\text{g}$  
> → $x = 4500 \pm 10$，即 $4490 \leq x \leq 4510$

### 相对误差

$$e_r(x^*) = \frac{x - x^*}{x}$$

实际计算中用：$e_r(x^*) \approx \frac{x - x^*}{x^*}$

**相对误差限 $\varepsilon_r^*$：** $\left|\frac{x - x^*}{x^*}\right| \leq \varepsilon_r^*$，常用 $\varepsilon_r^* = \frac{\varepsilon^*}{|x^*|}$

> **什么时候用相对误差？** 测 1000m 差 1m 和测 1m 差 1mm，绝对误差无法比较，必须用相对误差。

---

## 1.4 ⭐ 有效数字（高频考点）

**通俗理解：** 四舍五入得到的近似值，误差不超过末位半个单位。

**严格定义：** 将 $x^*$ 写成规格化形式：

$$x^* = \pm 0.a_1a_2\cdots a_n \cdots \times 10^p \quad (a_1 \neq 0)$$

若满足：

$$|x - x^*| \leq \frac{1}{2} \times 10^{p-n}$$

则称 $x^*$ 有 **n 位有效数字**。

> 🌰 **例题：** $\pi = 3.14159265...$
>
> | 近似值 | 规格化 | $p$ | 误差 | 有效数字位数 |
> |--------|--------|:---:|------|:---:|
> | $3$ | $0.3 \times 10^1$ | 1 | $0.14159... \leq 0.5$ | 1位 |
> | $3.14$ | $0.314 \times 10^1$ | 1 | $0.00159... \leq 0.005$ | 3位 |
> | $3.1416$ | $0.31416 \times 10^1$ | 1 | $0.0000073... \leq 0.00005$ | 5位 |

> **结论：** 四舍五入法取准确值前 n 位 → 必有 n 位有效数字。

**做题套路：**
1. 把近似值写成 $0.xxx \times 10^p$ 形式
2. 算误差 $|x - x^*|$
3. 看误差 $\leq \frac{1}{2} \times 10^{p-n}$ 的最大 n 是多少

---

## 1.5 数值计算的四个注意事项

| 原则 | 原因 | 对策 |
|------|------|------|
| **避免相近数相减** | $e_r(x-y) = \frac{e(x)-e(y)}{x-y}$，分母极小→误差巨大 | 变换公式 |
| **避免大数吃小数** | 对阶时小数被吞掉 | 改求和/计算顺序 |
| **避免小分母** | 误差被放大 | 避免或变换 |
| **提高效率** | 减少运算次数 | **秦九韶算法** |

> 🌰 **大数吃小数经典题：** 8位小数计算机求 $x^2 - (10^9+1)x + 10^9 = 0$
> - 精确解：$x_1 = 10^9$，$x_2 = 1$
> - 直接用求根公式 → 对阶时 $10^9 + 1 \approx 10^9$ → $x_2$ 算出为 0 ❌
> - **修正：** 用 $x_2 = \frac{2c}{-b+\sqrt{b^2-4ac}}$ 替代，得到正确结果 1 ✅

### 秦九韶算法

多项式：$P_n(x) = a_0x^n + a_1x^{n-1} + \cdots + a_n$

改写：$P_n(x) = (\cdots(a_0x + a_1)x + a_2)x + \cdots + a_{n-1})x + a_n$

递推：$b_0 = a_0$，$b_k = b_{k-1}x + a_k \;\;(k=1,2,...,n)$

| 方法 | 乘法次数 |
|------|:---:|
| 直接计算 | $\frac{n(n+1)}{2}$ |
| 秦九韶 | **$n$ 次** |

# <a id="ch2"></a>第2章 函数插值

## 2.1 插值是干什么的

> **已知：** n+1 个点 $(x_0,y_0), (x_1,y_1), ..., (x_n,y_n)$，$x_i$ 互不相同  
> **目标：** 找一个 n 次多项式 $P(x)$，满足 $P(x_i) = y_i$（穿过所有点）  
> **用途：** 用 $P(x)$ 估计任意 $x$ 处的函数值

**插值区间：** $[a,b]$ 包含所有插值节点

**多项式插值：** $P(x)$ 是 n 次代数多项式 $P(x) = a_0 + a_1x + \cdots + a_nx^n$

---

## 2.2 ⭐ Lagrange 插值（基函数法）

### 核心思想

给每个已知点配一个**基函数** $l_i(x)$：

$$l_i(x_j) = \begin{cases} 1, & i = j \\ 0, & i \neq j \end{cases}$$

### 基函数构造

$$l_i(x) = \prod_{j=0, j\neq i}^{n} \frac{x - x_j}{x_i - x_j}$$

> **记忆口诀：** 分子 = $(x - 别人的x)$ 全乘起来，分母 = $(自己x - 别人的x)$ 全乘起来

### 插值多项式

$$L_n(x) = \sum_{i=0}^{n} y_i \cdot l_i(x) = y_0l_0(x) + y_1l_1(x) + \cdots + y_nl_n(x)$$

### 特例

| 节点数 | 次数 | 名称 | 基函数 |
|:---:|:---:|------|------|
| 2个点 | 1次 | **线性插值** | $l_0 = \frac{x-x_1}{x_0-x_1},\; l_1 = \frac{x-x_0}{x_1-x_0}$ |
| 3个点 | 2次 | **抛物线插值** | 三个二次基函数 |

---

## 2.3 ⭐ Newton 插值（差商法）

### 核心优势：承袭性

> 增加新节点时，Lagrange 要全部重算，Newton 只需**加一项**！⬅️ 考点

### 差商定义

- **一阶差商：** $f[x_0,x_1] = \frac{f(x_1)-f(x_0)}{x_1-x_0}$ （就是两点间斜率）
- **二阶差商：** $f[x_0,x_1,x_2] = \frac{f[x_1,x_2]-f[x_0,x_1]}{x_2-x_0}$
- **n阶差商：** $f[x_0,...,x_n] = \frac{f[x_1,...,x_n]-f[x_0,...,x_{n-1}]}{x_n-x_0}$

### 插值多项式

$$N_n(x) = f(x_0) + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + \cdots$$

### 差商计算表（必会）

> 🌰 **例题：** 已知点 $(0,0), (60,0.45), (120,3.6), (180,12.15)$

| $x$ | $f(x)$ | 一阶差商 | 二阶差商 | 三阶差商 |
|:---:|:---:|------|------|------|
| 0 | 0 | | | |
| | | $\frac{0.45-0}{60-0}=0.0075$ | | |
| 60 | 0.45 | | $\frac{0.0525-0.0075}{120-0}=3.75\times10^{-4}$ | |
| | | $\frac{3.6-0.45}{120-60}=0.0525$ | | $\frac{7.5\times10^{-4}-3.75\times10^{-4}}{180-0}=\frac{1}{480000}$ |
| 120 | 3.6 | | $\frac{0.1425-0.0525}{180-60}=7.5\times10^{-4}$ | |
| | | $\frac{12.15-3.6}{180-120}=0.1425$ | | |
| 180 | 12.15 | | | |

结果：$N(x) = 0 + 0.0075x + 3.75\times10^{-4}x(x-60) + \frac{1}{480000}x(x-60)(x-120)$

展开即为 $\frac{x^3}{480000}$，与 Lagrange 结果一致。

---

## 2.4 Lagrange vs Newton

| 对比维度 | Lagrange | Newton |
|------|----------|--------|
| 构造方式 | 基函数线性组合 | 差商逐项累加 |
| 增加新节点 | 全部重算 ❌ | **只加一项** ✅ |
| 需要差商表 | 不需要 | 需要 |
| 结果 | 相同 | 相同 |

# <a id="ch3"></a>第3章 数值积分

## 3.1 为什么需要数值积分

遇到以下情况，Newton-Leibniz 公式无法直接使用：

1. $f(x)$ 没有初等原函数，如 $\frac{\sin x}{x}$、$e^{-x^2}$
2. $f(x)$ 只有离散点数据，没有解析表达式
3. $f(x)$ 有原函数但表达式极复杂

---

## 3.2 机械求积与代数精度

### 机械求积公式

$$\int_a^b f(x)dx \approx \sum_{k=0}^{n} A_k f(x_k)$$

- $x_k$：求积节点
- $A_k$：求积系数（权），只与节点选取有关

### ⭐ 代数精度（高频考点）

> **定义：** 求积公式对 $f(x)=1, x, x^2, ..., x^m$ 精确成立，但对 $x^{m+1}$ 不成立，则具有 **m 次代数精度**。

**验证方法：** 依次代入 $1, x, x^2, ...$，看等式是否成立。

> 🌰 **例题：** 验证 $\int_{-1}^1 f(x)dx \approx f(-\frac{1}{\sqrt{3}}) + f(\frac{1}{\sqrt{3}})$ 的代数精度
> - 代入 $f(x)=1$：左边=2，右边=2 ✓
> - 代入 $f(x)=x$：左边=0，右边=0 ✓
> - 代入 $f(x)=x^2$：左边=$\frac{2}{3}$，右边=$\frac{2}{3}$ ✓
> - 代入 $f(x)=x^3$：左边=0，右边=0 ✓
> - 代入 $f(x)=x^4$：左边=$\frac{2}{5}$，右边=$\frac{2}{9}$ ✗
> → 代数精度为 **3**

---

## 3.3 ⭐ Newton-Cotes 公式（必背）

### 三个核心公式

| 名称 | n | 公式 | 代数精度 |
|------|:---:|------|:---:|
| **梯形公式** | 1 | $T = \frac{b-a}{2}[f(a)+f(b)]$ | 1 |
| **Simpson公式** | 2 | $S = \frac{b-a}{6}[f(a)+4f(\frac{a+b}{2})+f(b)]$ | 3 |
| **Cotes公式** | 4 | $C = \frac{b-a}{90}[7f_0+32f_1+12f_2+32f_3+7f_4]$ | 5 |

> 📌 **Simpson 公式最重要！** 三个点达到三次代数精度。

### 通用形式

$$\int_a^b f(x)dx \approx (b-a)\sum_{k=0}^{n} C_k^{(n)} f(x_k)$$

其中 $C_k^{(n)}$ 为 Cotes 系数（可查表）。

---

## 3.4 ⭐ 复合求积公式（提高精度）

### 思想

把 $[a,b]$ 等分 n 份，每份用低阶公式，求和。

### 复合梯形公式

步长 $h = \frac{b-a}{n}$，节点 $x_i = a + ih$

$$T_n = \frac{h}{2}\left[f(a) + 2\sum_{i=1}^{n-1}f(x_i) + f(b)\right]$$

### ⭐ 复合 Simpson 公式（n 为偶数）

$$S_n = \frac{h}{6}\left[f(a) + 4\sum_{i=0}^{n-1}f(x_{i+1/2}) + 2\sum_{i=1}^{n-1}f(x_i) + f(b)\right]$$

> **记忆：** $h/6 \times$ [端点 + **4×中点** + **2×内点** + 另一端点]

---

## 3.5 ⭐ Gauss 求积公式

### 什么是 Gauss 点

> 如果插值型求积公式具有 **2n+1** 次代数精度，则节点称为 **Gauss 点**，公式称为 **Gauss 求积公式**。

### 两点 Gauss-Legendre 公式

$$\int_{-1}^{1} f(x)dx \approx f\left(-\frac{\sqrt{3}}{3}\right) + f\left(\frac{\sqrt{3}}{3}\right)$$

### 任意区间 $[a,b]$ 的变换

$$x = \frac{b-a}{2}t + \frac{a+b}{2}$$

$$\int_a^b f(x)dx = \frac{b-a}{2}\int_{-1}^{1} f\left(\frac{b-a}{2}t + \frac{a+b}{2}\right)dt$$

**使用步骤：**
1. 做变量变换将 $[a,b]$ 变到 $[-1,1]$
2. 查表得 Gauss 点和系数
3. 代入求积公式

# <a id="ch4"></a>第4章 线性方程组求解

## 4.1 两类方法概览

| 直接法 | 迭代法 |
|------|------|
| 有限步得精确解（不计舍入误差） | 逐次逼近，满足精度后停止 |
| 适用于小型稠密矩阵 | 适用于**大型稀疏矩阵** |
| 高斯消去法、**LU分解** | **Jacobi、Gauss-Seidel** |

---

## 4.2 ⭐ 高斯消去法 → LU 分解

### 高斯消去法过程

```
Ax = b  →  消元（化为上三角）  →  回代求解
```

### ⭐ LU 分解（Doolittle 分解）

> 将 $A$ 分解为 $A = LU$：
> - $L$：**单位下三角矩阵**（对角线全为 1）
> - $U$：**上三角矩阵**

### 紧凑格式计算（必会动手算）

**计算顺序：** 从外到内，先算行（$U$），再算列（$L$）

**递推公式：**

$$u_{rj} = a_{rj} - \sum_{k=1}^{r-1} l_{rk}u_{kj} \quad (j = r, r+1, ..., n)$$

$$l_{ir} = \frac{a_{ir} - \sum_{k=1}^{r-1} l_{ik}u_{kr}}{u_{rr}} \quad (i = r+1, ..., n)$$

### 方程组求解两步走

1. **前代：** $Ly = b$（求 $y$）
2. **回代：** $Ux = y$（求 $x$）

---

## 4.3 ⭐ 迭代法

### 矩阵分裂

$$A = D - L - U$$

- $D$：对角线矩阵
- $-L$：严格下三角部分
- $-U$：严格上三角部分

### Jacobi 迭代

**迭代格式：**

$$x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j \neq i} a_{ij}x_j^{(k)}\right)$$

**矩阵形式：**
$$x^{(k+1)} = D^{-1}(L+U)x^{(k)} + D^{-1}b$$

### ⭐ Gauss-Seidel 迭代（收敛更快）

**迭代格式：**

$$x_i^{(k+1)} = \frac{1}{a_{ii}}\left(b_i - \sum_{j < i} a_{ij}x_j^{(k+1)} - \sum_{j > i} a_{ij}x_j^{(k)}\right)$$

> **关键区别：** G-S 使用最新值！$j<i$ 用 $x_j^{(k+1)}$，$j>i$ 用 $x_j^{(k)}$

**矩阵形式：**
$$x^{(k+1)} = (D-L)^{-1}Ux^{(k)} + (D-L)^{-1}b$$

### 收敛性

> **充要条件：** 迭代矩阵 $G$ 的谱半径 $\rho(G) < 1$

- Jacobi 迭代矩阵：$G_J = D^{-1}(L+U)$
- G-S 迭代矩阵：$G_{GS} = (D-L)^{-1}U$

> **注意：** G-S 不一定比 Jacobi 收敛快，甚至可能出现 Jacobi 收敛但 G-S 发散的情况！

# <a id="ch5"></a>第5章 非线性方程求根

## 5.1 基本概念

> 求 $f(x) = 0$ 的根，$f(x)$ 为非线性函数。

**单根：** $f(x^*) = 0$ 但 $f'(x^*) \neq 0$

**重根（k 重）：** $f(x^*) = f'(x^*) = \cdots = f^{(k-1)}(x^*) = 0$，但 $f^{(k)}(x^*) \neq 0$

---

## 5.2 二分法

### 条件

$f(x)$ 在 $[a,b]$ 连续，且 $f(a)f(b) < 0$

### 步骤

1. 取中点 $x = \frac{a+b}{2}$
2. 若 $f(x)=0$，则 $x$ 为根
3. 若 $f(a)f(x) < 0$，则 $b = x$；否则 $a = x$
4. 重复直到 $|b-a| < \varepsilon$

### 误差估计

$$|x^* - x_n| \leq \frac{b-a}{2^{n+1}}$$

### 优缺点

| 优点 | 缺点 |
|------|------|
| 简单、一定收敛 | 收敛速度慢（线性） |
| 误差容易估计 | 只能求奇数重实根 |

---

## 5.3 不动点迭代法

### 基本思想

将 $f(x)=0$ 改写为 $x = \varphi(x)$，构造迭代：

$$x_{n+1} = \varphi(x_n)$$

若 $\{x_n\}$ 收敛到 $x^*$，且 $\varphi$ 连续，则 $x^*$ 是 $f(x)=0$ 的根。

### ⭐ 收敛条件

> **局部收敛定理：** 若 $x^*$ 是 $\varphi$ 的不动点，$|\varphi'(x^*)| < 1$，则迭代法**局部收敛**。

- $|\varphi'(x^*)| < 1$ → 收敛
- $|\varphi'(x^*)| > 1$ → 发散
- $|\varphi'(x^*)|$ 越小，收敛越快

---

## 5.4 ⭐ 牛顿迭代法（最重要！）

### 公式

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

### 几何意义：切线法

过 $(x_n, f(x_n))$ 作曲线的切线，切线与 $x$ 轴的交点即为 $x_{n+1}$。

### 特点

- **平方收敛**（收敛速度很快）
- 需要计算导数 $f'(x)$
- 要求 $f'(x) \neq 0$

### 算法步骤

1. 输入初值 $x_0$、误差限 $\varepsilon$、最大迭代次数 $N$
2. 计算 $x_{n+1} = x_n - f(x_n)/f'(x_n)$
3. 若 $|x_{n+1} - x_n| < \varepsilon$，输出 $x_{n+1}$；否则继续

---

## 5.5 弦截法（不用求导）

### 公式

$$x_{n+1} = x_n - \frac{f(x_n)(x_n - x_{n-1})}{f(x_n) - f(x_{n-1})}$$

> **思想：** 用割线斜率代替切线斜率 → $f'(x_n) \approx \frac{f(x_n)-f(x_{n-1})}{x_n-x_{n-1}}$

### 与牛顿法对比

| | 牛顿法 | 弦截法 |
|------|:---:|:---:|
| 需要导数 | ✅ | ❌ |
| 初值数量 | 1个 | **2个** |
| 收敛速度 | 平方收敛 | 超线性收敛（稍慢） |

---

## 5.6 三种方法对比

| 方法 | 公式 | 收敛速度 | 条件 |
|------|------|:---:|------|
| 二分法 | $x = \frac{a+b}{2}$ | 线性 | $f(a)f(b)<0$ |
| 不动点迭代 | $x_{n+1} = \varphi(x_n)$ | 线性 | $\vert\varphi'(x^*)\vert<1$ |
| 牛顿法 | $x_{n+1} = x_n - \frac{f}{f'}$ | **平方** | $f' \neq 0$ |
| 弦截法 | $x_{n+1} = x_n - \frac{f(x_n)(x_n-x_{n-1})}{f(x_n)-f(x_{n-1})}$ | 超线性 | 两个初值 |

---

# <a id="ch6"></a>第6章 常微分方程数值解法

## 6.1 初值问题

$$\begin{cases} \frac{dy}{dx} = f(x,y), & a \leq x \leq b \\ y(x_0) = y_0 \end{cases}$$

**离散化方法：** 将连续区间分成 $N$ 等份，步长 $h$，逐点求近似值。

---

## 6.2 Euler 法（最简单）

### 显式 Euler（向前差商）

$$y_{n+1} = y_n + hf(x_n, y_n)$$

- **几何意义：** 用当前点的切线方向走一步
- **局部截断误差：** $O(h^2)$，**1阶精度**

### 隐式 Euler（向后差商）

$$y_{n+1} = y_n + hf(x_{n+1}, y_{n+1})$$

- 需要解方程（$y_{n+1}$ 出现在两边）

---

## 6.3 ⭐ 改进的 Euler 法（2阶）

### 梯形公式

$$y_{n+1} = y_n + \frac{h}{2}[f(x_n,y_n) + f(x_{n+1},y_{n+1})]$$

### 预测-校正格式（实际使用）

**预测（Euler步）：**
$$\tilde{y}_{n+1} = y_n + hf(x_n, y_n)$$

**校正（梯形步）：**
$$y_{n+1} = y_n + \frac{h}{2}[f(x_n,y_n) + f(x_{n+1},\tilde{y}_{n+1})]$$

**等价 K 形式：**

$$\begin{cases} K_1 = f(x_n, y_n) \\ K_2 = f(x_n+h, y_n+hK_1) \\ y_{n+1} = y_n + \frac{h}{2}(K_1 + K_2) \end{cases}$$

- **局部截断误差：** $O(h^3)$，**2阶精度**

---

## 6.4 ⭐⭐⭐ 四阶 Runge-Kutta 法（最重要！）

$$y_{n+1} = y_n + \frac{h}{6}(K_1 + 2K_2 + 2K_3 + K_4)$$

$$\begin{cases} K_1 = f(x_n, y_n) \\ K_2 = f\left(x_n + \frac{h}{2},\; y_n + \frac{h}{2}K_1\right) \\ K_3 = f\left(x_n + \frac{h}{2},\; y_n + \frac{h}{2}K_2\right) \\ K_4 = f\left(x_n + h,\; y_n + hK_3\right) \end{cases}$$

> **记忆口诀：** $h/6 \times (K_1 + 2K_2 + 2K_3 + K_4)$，系数是 1-2-2-1

**$K_i$ 的物理意义：**
- $K_1$：起点斜率
- $K_2$：中点斜率（用 $K_1$ 估计）
- $K_3$：中点斜率（用 $K_2$ 重新估计）
- $K_4$：终点斜率（用 $K_3$ 估计）

**局部截断误差：** $O(h^5)$，**4阶精度** ← 最常用！

---

## 6.5 方法总结

| 方法 | 核心公式 | 阶数 | 每步计算次数 |
|------|------|:---:|:---:|
| 显式Euler | $y_{n+1} = y_n + hf_n$ | 1阶 | 1次 |
| 隐式Euler | $y_{n+1} = y_n + hf_{n+1}$ | 1阶 | 1次（需解方程） |
| 梯形公式 | $y_{n+1} = y_n + \frac{h}{2}(f_n+f_{n+1})$ | 2阶 | 2次（隐式） |
| **改进Euler** | 预测 $\tilde{y} = y_n + hf_n$，校正用梯形 | **2阶** | 2次 |
| **四阶R-K** | $y_{n+1} = y_n + \frac{h}{6}(K_1+2K_2+2K_3+K_4)$ | **4阶** | **4次** |

> 📌 **考试重点：** 四阶 Runge-Kutta 公式必背，能用手算一两步。

# 📋 各章节公式速查卡

## 第1章 引论
- 绝对误差：$e = x - x^*$
- 相对误差：$e_r = \frac{x-x^*}{x^*}$
- 有效数字：$|x-x^*| \leq \frac{1}{2} \times 10^{p-n}$

## 第2章 函数插值
- Lagrange：$L_n(x) = \sum y_i \cdot l_i(x)$，$l_i(x) = \prod_{j\neq i} \frac{x-x_j}{x_i-x_j}$
- Newton：$N_n(x) = f(x_0) + \sum_{k=1}^n f[x_0,...,x_k]\prod_{j=0}^{k-1}(x-x_j)$

## 第3章 数值积分
- 梯形：$T = \frac{b-a}{2}[f(a)+f(b)]$
- Simpson：$S = \frac{b-a}{6}[f(a)+4f(\frac{a+b}{2})+f(b)]$
- 复合梯形：$T_n = \frac{h}{2}[f(a)+2\sum f(x_i)+f(b)]$
- 代数精度检验：代入 $1, x, x^2, ...$

## 第4章 方程组
- LU：$A = LU$，先算行(U)再算列(L)，$Ly=b$ → $Ux=y$
- Jacobi：$x_i^{(k+1)} = \frac{1}{a_{ii}}(b_i - \sum_{j\neq i}a_{ij}x_j^{(k)})$
- G-S：$x_i^{(k+1)} = \frac{1}{a_{ii}}(b_i - \sum_{j<i}a_{ij}x_j^{(k+1)} - \sum_{j>i}a_{ij}x_j^{(k)})$

## 第5章 非线性方程
- 二分法：$x = \frac{a+b}{2}$
- 牛顿法：$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$
- 弦截法：$x_{n+1} = x_n - \frac{f(x_n)(x_n-x_{n-1})}{f(x_n)-f(x_{n-1})}$

## 第6章 常微分方程
- Euler：$y_{n+1} = y_n + hf(x_n,y_n)$
- 改进Euler：预测 $\tilde{y}_{n+1} = y_n + hf_n$，校正 $y_{n+1} = y_n + \frac{h}{2}(f_n+f_{n+1})$
