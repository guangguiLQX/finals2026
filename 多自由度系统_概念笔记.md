## 一、为什么要学多自由度？

单自由度（前面所有内容）只能描述**一个**振动模式。

真实工程结构——汽车车身、飞机机翼、桥梁——需要多个坐标才能描述所有可能的振动形态。

核心思想：**n 自由度系统 → n 个固有频率 → n 个振动模态 → 任何振动都是模态的叠加**

## 二、运动方程（第11讲）

### 2.1 矩阵形式

```
M ẍ + K x = 0
```

- **M**：质量矩阵（n×n），正定对称
- **K**：刚度矩阵（n×n），对称（可能半正定）
- **x**：位移向量，[x₁, x₂, ..., xₙ]ᵀ

### 2.2 刚度矩阵 K —— 怎么得到？

**定义**：`k_ij` = 让第 j 个自由度发生**单位位移**（其余自由度位移=0），在第 i 个自由度上需要施加的力。

**操作（单位位移法）**：
```
Step 1：令 xⱼ = 1，其他 x = 0
Step 2：画出此时各弹簧的变形
Step 3：对每个质量列平衡方程，求出各自由度所需的外力
Step 4：这些外力就是 K 的第 j 列

对每个 j 重复 → 得到完整的 K
```

### 2.3 二自由度平动系统的标准形式

```
m₁  m₂ 由弹簧 k₁, k₂, k₃ 连接（两端可连固定端）

│ m₁   0  ││ẍ₁│     │ k₁+k₂   -k₂  ││x₁│     │0│
│ 0    m₂ ││ẍ₂│  +  │  -k₂   k₂+k₃ ││x₂│  =  │0│
```

> **规律**：对角元 = 连接该质量的所有弹簧刚度之和  
> 非对角元 = 负的连接两质量的弹簧刚度（k_ij = k_ji = -k_connecting）

### 2.4 扭转振动系统

形式完全一样，只需替换：
```
x → θ（角位移）
m → I（转动惯量）
k → k_t（扭转刚度）
F → M（力矩）
```

### 2.5 耦合

矩阵非对角元 ≠ 0 → 自由度之间存在耦合：

**静力耦合（刚度耦合）**：K 非对角
**动力耦合（惯性耦合）**：M 非对角

物理表现：推一个质量，别的质量也跟着动。

## 三、柔度法（第12讲）

### 3.1 刚度法的"镜像"

**刚度法**：先定位移，求力 → K  
**柔度法**：先定力，求位移 → F = K⁻¹

**柔度影响系数 f_ij**：在第 j 个自由度施加**单位力**，在第 i 个自由度产生的位移。

### 3.2 柔度法运动方程

```
x = F(P - Mẍ)

整理：FMẍ + x = FP

×K：Mẍ + Kx = P   ← 回到刚度形式
```

### 3.3 什么时候用柔度法？

柔度法特别适合**用材料力学公式求位移**的系统——比如简支梁上的质量。

**例子：简支梁上两个集中质量**
```
简支梁长 L，抗弯刚度 EJ
两个质量位于 a 和 b 处

柔度系数用材料力学公式：
f_ab = a·b·(L² - a² - b²)/(6·EJ·L)

得到 F = │f₁₁  f₁₂│
         │f₂₁  f₂₂│

K = F⁻¹
```

### 3.4 刚度法 vs 柔度法

| | 刚度法 | 柔度法 |
|---|-------|-------|
| 出发点 | 单位位移 → 力求 K | 单位力 → 位移求 F |
| K 好求？ | 弹簧-质量系统易求 | 梁、框架等连续结构难求 |
| 操作 | 固定其他自由度位移=0 | 施加单位力 |
| 关系 | K | F = K⁻¹ |

## 四、拉格朗日方程（第12讲补充）

当系统复杂（有几何约束、转动+平动耦合），用拉格朗日方程更优雅。

### 4.1 拉格朗日函数

```
L = T - V

T：系统总动能
V：系统总势能
```

### 4.2 拉格朗日方程

对每个广义坐标 q_i：
```
d  ┌ ∂L  ┐   ∂L
── │ ─── │ - ─── = Q_i      (i = 1~n)
dt └ ∂q̇_i┘   ∂q_i
```

其中 Q_i 是非保守力（阻尼力、外力）对应的广义力。对于自由振动，Q_i = 0。

### 4.3 势能函数 V 的符号注意

在拉格朗日力学中：
```
势能 V 增加 → L = T-V 减小 → 产生恢复力
```

对有重力的系统，`V = ½kx² + mgh`（取向上为正时 h 增则 V 增）。

最终结果和牛顿法完全一致，只是推导路径不同。

## 五、特征值问题——求固有频率和振型（第13讲）

### 5.1 设解

假设所有自由度以同样的频率 ω 振动：
```
x(t) = φ · sin(ωt + ψ)
```

代入 `Mẍ + Kx = 0`：
```
(-ω²M + K)φ = 0

即：(K - ω²M) φ = 0     ← 广义特征值问题
```

### 5.2 特征方程

非零解条件（系数矩阵奇异）：
```
det(K - ω²M) = 0        ← 特征方程
```

展开得 ω²（或 ω^(2n)）的 n 次多项式，解出 n 个特征值：
```
ω₁² < ω₂² < ... < ωₙ²      （从小到大排列）

ω₁：第一阶固有频率（基频）
ω₂：第二阶固有频率
...
```

### 5.3 求模态振型 φ⁽ⁱ⁾

把每个 ω_i² 代回：
```
(K - ω_i²M) φ⁽ⁱ⁾ = 0
```

解出 φ⁽ⁱ⁾（非唯一，有一个自由度的比例可任选）。

**φ⁽ⁱ⁾ 的含义**：系统以第 i 阶固有频率 ω_i 振动时，各自由度振幅的**比例关系**。

> ⚠️ **φ⁽ⁱ⁾ 只定形状，不定绝对幅值。** 常归一化为：
> - 最大分量为 1
> - 或第一个分量为 1
> - 或模态质量为 1

### 5.4 伴随矩阵法求 φ

特征值处 `B = K - ω_i²M` 秩为 n-1（单根时），伴随矩阵 `adj(B)` 的秩为 1。

**任意非零列都是 φ⁽ⁱ⁾！** 选一列非零的即可。

### 5.5 模态的物理图像

**二自由度系统**：
```
φ⁽¹⁾ = {1, 1}ᵀ      第一阶：两个质量同向振动（in-phase）
φ⁽²⁾ = {1, -1}ᵀ     第二阶：两个质量反向振动（out-of-phase）
```

**规律**：
- 第1阶：所有自由度同向振动，无节点
- 第2阶：出现1个节点
- 第3阶：出现2个节点
- ...
- 第 n 阶：出现 n-1 个节点

## 六、模态正交性与解耦（第14讲）——多自由度的"魔法"

### 6.1 正交性

**不同阶模态之间**：
```
(φ⁽ⁱ⁾)ᵀ M φ⁽ʲ⁾ = 0    (i ≠ j)     M-正交
(φ⁽ⁱ⁾)ᵀ K φ⁽ʲ⁾ = 0    (i ≠ j)     K-正交
```

**证明思路**：
```
由 (K - ω_i²M)φ⁽ⁱ⁾ = 0  →  φ⁽ʲ⁾ᵀKφ⁽ⁱ⁾ = ω_i²·φ⁽ʲ⁾ᵀMφ⁽ⁱ⁾
由 (K - ω_j²M)φ⁽ʲ⁾ = 0  →  φ⁽ⁱ⁾ᵀKφ⁽ʲ⁾ = ω_j²·φ⁽ⁱ⁾ᵀMφ⁽ʲ⁾
相减：(ω_i² - ω_j²)·φ⁽ⁱ⁾ᵀMφ⁽ʲ⁾ = 0
当 ω_i ≠ ω_j → φ⁽ⁱ⁾ᵀMφ⁽ʲ⁾ = 0 → φ⁽ⁱ⁾ᵀKφ⁽ʲ⁾ = 0
```

### 6.2 模态质量与模态刚度

**同阶**：
```
m_pi = (φ⁽ⁱ⁾)ᵀ M φ⁽ⁱ⁾     ← 模态质量（标量）
k_pi = (φ⁽ⁱ⁾)ᵀ K φ⁽ⁱ⁾     ← 模态刚度（标量）

且：ω_i² = k_pi / m_pi   ← 形式和单自由度完全一样！
```

### 6.3 模态矩阵 Φ

把 n 个模态向量拼成矩阵：
```
Φ = [φ⁽¹⁾  φ⁽²⁾  ...  φ⁽ⁿ⁾]     （n×n）
```

**对角化性质**：
```
Φᵀ M Φ = diag(m_p1, m_p2, ..., m_pn) = M_p    ← 对角矩阵！
Φᵀ K Φ = diag(k_p1, k_p2, ..., k_pn) = K_p    ← 对角矩阵！
```

非对角元全部为零（正交性）。这就是"魔法"所在。

### 6.4 解耦——整个多自由度理论的终极目标

做坐标变换：
```
x(t) = Φ p(t)

物理坐标 x = 模态矩阵 × 模态坐标 p
```

代入 `Mẍ + Kx = 0`：
```
MΦp̈ + KΦp = 0
ΦᵀMΦp̈ + ΦᵀKΦp = 0     ← 左乘 Φᵀ
M_p p̈ + K_p p = 0       ← 对角矩阵！
```

**n 个耦合方程 → n 个独立的单自由度方程**：

```
m_pi · p̈_i(t) + k_pi · p_i(t) = 0      (i = 1~n)

即：p̈_i + ω_i² · p_i = 0
```

每个都像单自由度无阻尼自由振动一样解：
```
p_i(t) = p_i(0)cos(ω_i t) + (ṗ_i(0)/ω_i)sin(ω_i t)
```

最后叠加回物理响应：
```
x(t) = Σ φ⁽ⁱ⁾ · p_i(t) = Φ p(t)
```

### 6.5 初始条件的变换

物理坐标的初始条件要转到模态坐标：
```
p(0) = Φ⁻¹ x(0)

更实用的做法（利用正交性）：
p_i(0) = (φ⁽ⁱ⁾)ᵀ M x(0) / m_pi
ṗ_i(0) = (φ⁽ⁱ⁾)ᵀ M ẋ(0) / m_pi
```

### 6.6 归一化模态（让公式更简洁）

让每个模态的模态质量 = 1：
```
φ_N⁽ⁱ⁾ = φ⁽ⁱ⁾ / √(m_pi)

则：Φ_Nᵀ M Φ_N = I         ← 单位矩阵！
    Φ_Nᵀ K Φ_N = Λ         ← 特征值对角矩阵 = diag(ω₁², ..., ωₙ²)
```

此时初始条件变换也简化：
```
p_N(0) = Φ_Nᵀ M x(0)
ṗ_N(0) = Φ_Nᵀ M ẋ(0)
```

## 七、零根 / 刚体模态（第15讲）

### 7.1 什么时候出现零根？

```
det(K) = 0  →  K 奇异  →  存在 ω² = 0 的特征根
```

物理原因：系统**没有完全约束**，可以做刚体运动（整体平动或转动），不做弹性变形。

**例子**：
- 两端自由的弹簧-质量链 → 整体平动
- 自由飞行的飞行器 → 6个零根（3平动+3转动）
- 一个固定端的弹簧-质量 → 无零根（完全约束）

### 7.2 零频率的模态方程

```
ω₁ = 0  →  K φ⁽¹⁾ = 0
```

解 `K φ⁽¹⁾ = 0` 得到刚体模态。例如两端自由系统：`φ⁽¹⁾ = [1, 1, 1, 1]ᵀ`（整体平动）。

### 7.3 零根模态的动力学

对 ω₁ = 0：
```
m_p1 · p̈₁ + 0 · p₁ = 0
→ p̈₁ = 0
→ p₁(t) = a·t + b        ← 线性增长！不是振荡！
```

对应的物理响应：
```
x_rigid(t) = φ⁽¹⁾ · (a·t + b)
```

系统以恒定速度漂移（整体平动/转动），不做振荡。

### 7.4 多重零根

如果 `rank(K) = n - r`，有 r 个零根 → r 个刚体模态。

需要 Gram-Schmidt 正交化确保 r 个零频模态彼此 M-正交和 K-正交。

**PPT 例题**：正方形排列的四质量系统，存在 2 个零根（x 方向平动 + y 方向平动，或整体平动 + 整体转动）。

### 7.5 约束条件消除刚体模态

如果物理上不希望有刚体运动，施加约束：
```
(φ⁽¹⁾)ᵀ M x = 0
```
这确保响应中没有刚体模态分量。

## 八、多自由度受迫振动（概念要点）

加外力：
```
Mẍ + Cẋ + Kx = P(t)
```

解耦后：
```
M_p p̈ + C_p ṗ + K_p p = Φᵀ P(t)
```

如果阻尼矩阵 C 满足**比例阻尼**条件（C = αM + βK，即也可以被模态矩阵对角化），则 n 个模态方程完全独立。

否则（非比例阻尼）需要用**复模态理论**——超出本课范围。

## 九、完整解题流程（考试大题模板）

### 多自由度自由振动标准大题

```
已知：系统参数（质量、弹簧、几何）

Step 1：建立 M 和 K（单位位移法或柔度法或拉格朗日法）

Step 2：解特征方程 det(K - ω²M) = 0
        → ω₁², ω₂², ..., ωₙ²
        → ω₁, ω₂, ..., ωₙ

Step 3：对每个 ω_i，解 (K - ω_i²M)φ⁽ⁱ⁾ = 0
        → 模态振型 φ⁽¹⁾, φ⁽²⁾, ..., φ⁽ⁿ⁾

Step 4：验证正交性（可选）
        (φ⁽ⁱ⁾)ᵀMφ⁽ʲ⁾ = 0 (i≠j)

Step 5：由初始条件 x(0), ẋ(0)
        → 求模态坐标初始值 p_i(0), ṗ_i(0)
        → 解 p_i(t) = p_i(0)cos(ω_i t) + (ṗ_i(0)/ω_i)sin(ω_i t)

Step 6：叠加回物理坐标
        x(t) = Σ φ⁽ⁱ⁾·p_i(t)
```

## 十一、易错点

1. **刚度矩阵的对称性**：`k_ij = k_ji`，如果不对称肯定算错了
2. **单位位移法要固定其他位移=0**——不是力=0，是位移=0
3. **特征方程是 det(K - ω²M) = 0**，不是 det(M - ω²K)
4. **求模态时代入的是 ω_i²**，不是 ω_i
5. **模态只定形状不定幅值**——任何非零倍数都是同一个模态
6. **零根时 p₁(t) = at+b**，不是 sin/cos 振荡
7. **初始条件变换必须用正交性公式**，不能简单代入
8. **Gram-Schmidt 正交化**对重根是必须的步骤
9. **ω₁ 是最小的固有频率（基频）**，不要搞错大小顺序
10. **比例阻尼**：C = αM + βK 才可被模态矩阵对角化
